TC Route Finder Pilot Testing

zhou

March 15, 2010

After several months' work, Jess, Chandler, Shaoqing, Pranav, I, and some other colleagues have created an interactive map application of Teachers College. It has two interfaces: User Interface: http://jay.tc.columbia.edu/tcroutefinder/trunk/prototype/TCRouteFinder.html Anyone can use this interface to search for a route between any two rooms at TC. Admin Interface: http://jay.tc.columbia.edu/tcroutefinder/trunk/prototype/TCRouteMarker.html Admins can use this interface to download the XML database of the rooms and paths of TC, then modify or create new ones on the local computer. The modified XML can then be uploaded to the server to update the existing database. We have created a database of all the accessible TC floors and buildings except those being renovated. There are also some bugs with the UI and route search function which we are fixing. It is a good idea for us to collect some user feedback on this application now before launching it for the TC community. We will present this application at Tuesday's D&R meeting, and you can leave your feedback in this blog post. Please leave this post in private. Thanks.
