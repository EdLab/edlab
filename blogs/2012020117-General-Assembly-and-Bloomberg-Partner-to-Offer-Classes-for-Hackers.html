General Assembly and Bloomberg Partner to Offer Classes for 'Hackers'

Kate Meersschaert

February 1, 2012

Inspired by this "answer wiki on Quora", and in partnership with Bloomberg, General Assembly, has created a special 10-week certification course, comprised of a core of computer science concepts tailored to the self-taught "hacker" and anyone who identifies themself somewhere on the DIY CS spectrum. Some of the skills that the self-taught practitioner might lack as mentioned in the Quora post referenced above that inspired this course include: Data Structures and Patterns Compilers (implementations) Programming Languages (implementations) Machine Learning (implementations) Finite State Machines Code-as-Data Functional Programming Object Composition Recursion Lambda Calculus Pointers Type Systems, Category Theory Search Numerical methods Common vocabulary, jargon, and conventions Concurrency Learn more about this 10-week certification program here.
