Coding is fun? Practice coding skills with Coding Duels!

Sharon Hsiao

June 15, 2011

Not long ago, Microsoft launched a platform called, pex4fun. It's a social, community-based, intelligent, fun, analytics, programming languages learning platform. Some key functions of Pex4fun: -community based authoring; build on top of wiki -engage programming games; allow to create puzzles with correct implementation for duel challenges (such implementation can become an online IDE. Each step becomes the hint for users when they encounter problems) -social; live feeds to show duel results or puzzle challenges or see how other people solving the problems. -analytics; system provides the analytics results, for instructors to monitor the errors, attempts etc. -there's going to be a mobile version too. Assuming we are solving “sorting algorithm” problem, it's hard to be creative with the solutions. It must be sorted. Is solving it fun or competing socially makes it fun? So, how many gaming elements should we have within the learning platform?
