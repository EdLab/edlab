Project Idea: Interactive Textbooks

Jeannie Crowley

June 14, 2010

I would like to create a truly interactive textbook experience where the user's contributions are added to the text. I'm interested in creating one chapter of a science textbook as a proof-of-concept for how textbooks can be designed for mobile devices rather than migrated to mobile devices. I choose science as a topic because it lends itself well to user-generated content (data) on a device that does not support extensive typing. A few of my ideas:

  • Embed data collection tools in the text so users who perform experiments can submit their results.
  • The user needs to be able to add information when not connected to the Internet that will be synced when a wifi network is available (to overcome a common issue with the iPad).
  • Data collected by users will be displayed within the text and can be explored by region, date, etc.
  • Discussion forums around each data set will help users share experiences and issues with the experiments.
