Mobile Route finder

Shaoqing Niu

May 28, 2010

I designed and implemented a mobile version of TC Route Finder this week. The website for TCRF will be automatically redirected to the mobile version if you are using a mobile device. Currently it works well for iPhones and iPads. There might be some problems for other mobile devices such as the Blackberry.

The interface of this mobile route finder is really concise as you don't need to type anything to get the route:

The output is an ordered list which is easy for users to follow:

Users could report an error if the given directions were not able to lead them to their destinations.
