Code Ed - "Girls take a byte out of tech"

Michelle Delateur

August 2, 2010

CNN Money posted an interesting piece about Code Ed, a non-profit created to teach young girls computer science. The program was piloted at Girls Preparatory. They learn web building, java, and then more advanced languages. The goal is to teach girls computer science and increase women in computer tech jobs. It could be an interesting group to follow and learn from!
