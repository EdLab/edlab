2007 Microsoft Office Beta Version

Linda Bloom

July 18, 2006

Has anyone tried the beta version of the 2007 Microsoft Office that was released in June? Between now and the release of the latest version you can download or navigate the beta version through your web browser. My first observations show that the format is the same with some menu changes - more images and options in the toolbars. I also noticed that the "ctrl" keys may have been adjusted, as in Ctrl + S = save - this doesn't seem to work anymore. This beta version comes with a good set of tutorials that highlight the program changes. I'm still clicking through and deciding how much of an impact this will have to Microsoft users that have become comfortable with what they already have. Check it out. 2007 Microsoft Office
