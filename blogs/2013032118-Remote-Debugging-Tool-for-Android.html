Remote Debugging Tool for Android

zhou

March 21, 2013

Finally! The wait is over! Attention EdLab (semi-)frontend developers, from now on you can debug our web apps on Android devices with more ease via the Remote Debugging Tool. I used it today to figured out what was breaking the Vialogues video player on NLT. It's still not as handy as using Safari User Agent to debug iOS devices. But it's a huge step closer than before. After installing some software from Google, setting the compiling path on my machine, configuring the Android phone and Chrome browser, I could finally read the Javascript errors occurring on my Android browser. Two weeks ago to do the same thing I had to guess what was happening in that black box which made me hate my Android phone.
