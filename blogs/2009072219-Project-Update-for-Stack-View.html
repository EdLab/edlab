Project Update for Stack View

Jeff Goldenson

July 22, 2009

People talk of learning curves. When they say that, I always envision a daunting slope along the x,y axis. Stack View's felt a little different - kinda like I threw myself into the bottom of an empty kidney-shaped pool. Concrete curves all around me. And it's nighttime. So for the past month+ I've been fumbling around with software stitchers, photographing strategies, learning scant bits of "computational photography" and "computer vision" as needed. It's taken a while, but I'm starting to figure some things out. I'm excited to talk to you guys about what I've learned, show you some real simple tests, and talk about the race to the finish (and maybe even enlist a teammate if there's interest)... Talk Friday - Jeff
