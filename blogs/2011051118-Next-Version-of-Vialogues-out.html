Next Version of Vialogues out!!

pramnab1

May 11, 2011

The Vialogues team is proud to announce the release of an updated Vialogues with new features and minor bug fixes. Some of the features included in this release are: - Moderators, participants can now delete comments. - Pagination. - RSS and sitemap. - Improved My Vialogues tab for distinguishing vialogues you are moderating and vialogues you are a participant in. We're looking forward to people using these and other features in a variety of educational/commerical settings. Thanks to the entire Vialogues team in getting this release out: Gonzalo, Amelia, Josh, Faisal, Stephen, Yudan, Megha, Joann, Hui Soo, Angela and others. Some other awaited features which are in the pipeline for the upcoming release are:- 1) Notifications 2) Improved Search 3) Revamped Embedded Vialogues 4) Revamped Vialogues homepage 5) Categorization 6) LiveStream and Instant Chat Enjoy and keep the feedback coming!
