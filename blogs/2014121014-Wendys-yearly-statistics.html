Wendy's yearly statistics

wendy

December 10, 2014

This is my first blog I think. This is a statistical report of the amount of work I do during the year. This year covers 2014 only. Number of Books Retrieved- 366 Number of Books Reshelved- 509 Number of Books Rush Processed- 173 Number of Books Withdrawn- 61 Number of Books Processed- 1,398 Number of Books Added to the collection- 1,138 Number of Missing Books Recovered- 52 Number of Books Brought to Hold Shelf- 781 Number of Books Brought to Reserve Shelf- 489 Number of Books Brought to Sorting Shelves- 1,302 Number of ILL Books Brought down from Range 8- 150 Number of Cafe Items- 679 Number of YBP Books Processed 154 Number of Doc Del Searched- 646 Number of ILL Books Searched- 139 Number of Books Transferred- 898 These statistics will go up in the next two weeks, but not by much. The YBP books come in already processed. I don't have to do anything to them. Each new year I make a new spreadsheet for Retrievals and Reshelving. These are books returned that go to room 4, 5, and sub-basement. This way it is easier to manage the statistics. I also have a new spreadsheet for the monthly statistics. This way the numbers are right at hand and easier to manage.
