Create Your Own 3-D Pictures

Demetri Lales

February 3, 2012

A big branch of mathematics that isn't discussed much is the subject of Geometry, which is used in several fields of study like architecture, and structural engineering. Google has a product that could be a valuable resource in helping students learn more about the properties of 3 dimensional objects called SketchUp. SketchUp is a free tool that enables users to draft 3 dimensional objects, and create any structure they can imagine. Think of it as a Paint program, only 3 dimensional and with more features like sharing pictures, seeing you model through Google Earth, and much more. This program is a great tool for educators to use when teaching students about 3-d shapes and their properties. Potential for sySTEMic Change: SketchUp can be a useful tool for math classroom, and other subjects as well, check out the Vialogue below to see how. There are many other free resources like SketchUp for math educators that can be easily found online, some of which can be used in other non-STEM subjects as well. It's very beneficial for a student to see how mathematical concepts like geometry can crossover into other subjects. Let's keep crossing over, everyone.
