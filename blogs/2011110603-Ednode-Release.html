Ednode Release

Stephen Pratt

November 6, 2011

(keep private) Last Friday marked the release of the newest version of Ednode . Over the past couple months, we've focused development efforts on creating a more sophisticated search and recommendation platform. Here's a list of some of the newest features:

  • Faster, more efficient searches
  • More comprehensive user recommendations
  • List of common profile terms generated for all user recommendations
  • Recommendations scored based on relevance to user
  • Users given the ability remove recommendations
  • Bookmarks allow a user to flag another users profile for later viewing
  • Easier profile edit access for external applications like Pundit
  • Use data is collected to help inform future development and improve recommendations
  • A cleaner UI with more intuitive controls
So what's next for Ednode? Well we've got a lot of data on our hands and it's time to start putting it to good use. Ednode already has a special role as Pundit's dedicated user information center, and this release's set of improvements gives it even more potential. By clarifying our API's and developing our recommendations algorithm further, we hope to turn Ednode into the definitive user information center across the full suite of EdLab applications. Grouping Ednode users into related cliques (something this new release makes possible) will allow us to provide more intelligent content recommendations on Pundit and Vialogues alike. Of course the entire Ednode team will need as much commentary and feedback as the lab can possibly offer. We welcome any questions, comments, or criticisms regarding the new release both here on the blog and in person at the lab. Also, next time you run into Megha, Basak, or Pranav, make sure to congratulate them on all the hard work they put into making this release happen. Here's to you guys; thanks for everything!
