Knowledge Sharing is here

pramnab1

November 4, 2011

*** PLEASE KEEP PRIVATE Today marks the start of the Development & Research Learning Series, a voluntary, weekly (or on-demand) meeting of staff to learn about a topic (e.g., HTML, CSS, Selenium, SAS, web analytics) that will help us grow professionally and intellectually. We imagine that each session will be led by one or two staff members and participants will be engaged in interactive activities as part of the learning experience. The meetings will take place every Friday from 4-5pm in the back conference room. We've set up a mSchool course to support the Development & Research Learning Series. You can access it at: http://library.tc.columbia.edu/moodle/course/view.php?id=1167 If you don't have a Moodle account you will need to create one. The Enrollment Key is: pranav The idea is that this course will serve as a permanent resource for all current and future staff. We can also take each module of the Development & Research Learning Series course and develop more in-depth mSchool courses (e.g., Using Excel for Data Analysis: http://library.tc.columbia.edu/moodle/course/view.php?id=1139). Today, I will be leading a session entitled: - Introduction to HTML by Pranav Garg If you are interested please RSVP to this blog post so I can plan accordingly. And if you have feedback, please let us know as well.
