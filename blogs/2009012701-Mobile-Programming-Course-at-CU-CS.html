Mobile Programming Course at CU-CS

Anthony Cocciolo

January 27, 2009

As Wednesday's EdLab seminar would indicate, it looks like a lot of people are interested developing applications for mobile phones (particularly the iPhone and Android). The chair of my program asked me about the possibilities of offering a class on this, and I was less than reassuring (I was concerned that the technical skills needed at this time are much higher than what TC students would possess, especially compared to other platforms such as web applications that use scripting languages). However, it looks like the Columbia Computer Science program is offering a course on just this: http://www.cs.columbia.edu/~nieh/teaching/e6998/ Since developing for these platforms is technically involved (as the prerequisites indicate), a course like this is probably best situated at CU CS (although one day, I wouldn't be surprised if TC did offer something in the CCTE program once it becomes more feasible for non-CS types to do this type of development).
