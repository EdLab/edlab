Host A Free Wordpress Blog With Amazon EC2

zhou

November 28, 2011

During the Thanksgiving weekend I took on a small project in setting up a Wordpress blog for my son Eugene. The blog is hosted with Amazon EC2 (Elastic Compute Cloud) and Route 53 services. Basically EC2 is Amazon's web hosting service with MySQL support. And Route 53 is a Domain Name Service (DNS). Amazon is very nice to provide these services for free in the first year when the usage is within the free tier, and at very low costs for just how much bandwidth is actually consumed when it's beyond the free threshold. So it's a perfect solution for small businesses which are still in the development stage. In my case, I purchased a .info domain for $1.99 a year from Godaddy.com and use Amazon's Route 53 to host the domain. I am not highly knowledgeable in Linux server and domain setup. But it was surprisingly not that difficult to get the EC2 virtual server running at the elastic IP address that Amazon provided, install the PHP and MySQL running environments, and install the Wordpress CMS. There are many tutorials and knowledge sharing you can find in Google. The most helpful two that I referred to are: Alexkorn and I Love Free Software. The aforementioned "I Love Free Software" used a template of Wordpress that is shared to all Amazon EC2 users to install the CMS conveniently if you don't want to spend the time to setup everything from scratch. But I setup everything from scratch myself because I want to control what are installed on my virtual server. The difficult part was connecting the Amazon IP with the domain name I bought from Godaddy. There was minimum tutorials or documentation found online on either Amazon or Godaddy, as Route 53 is relatively new. Also, I spent some time to figure out how to use Putty, Aptana, and WinSCP with Amazon EC2. By Saturday night I have got everything straight so that I can develop the Wordpress instance in the same working environment as the one I do EdLab works. If anyone is interested I am glad to share my experience. My next project idea is to setup the Amazon CloudFront service (the one we use to host Vialogues videos) so as to use videos in Wordpress. Wordpress doesn't allow uploading videos to its media library by default. But I think it would be very fun to upload both videos and pictures using the Wordpress publishing app on iPhone and Android.
