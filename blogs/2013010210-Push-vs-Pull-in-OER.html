"Push" vs. "Pull" in OER

Kate Meersschaert

January 2, 2013

Much has been written lately regarding the value of open educational resources (OER) to support online learning and MOOCs. However, this is the first article I have found that provides a cautionary note, re: their use for independent student learning. Instead, of the usual "push" approach, in which educators select which OER resources to include in their online courses, the authors make a case for a "pull" model. This model, in which learners select their own OER resources, is in direct opposition to the current MOOC movement, or even an LMS like Moodle, in which all discussion boards and OER resources like Wikis are pre-determined. Do you find the "pull" model to be more compelling? Are the authors right? Will this more learner-centric approach increase learning?
