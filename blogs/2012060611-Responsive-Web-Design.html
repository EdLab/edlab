Responsive Web Design

zhou

June 6, 2012

I was reading articles about Wordpress site design and implementation when I encountered this presentation about responsive web design. The idea is to make use of fluid width in defining dimensions of HTML elements so that they auto adapt to different screen sizes such as computers, iPad, and iPhone. The link to the presentation is here. I'd like to share it because it's very informing and helpful to our web projects. This presentation was given in the Wordpress NYC Meetup.
