CargoBot

Demetri Lales

December 2, 2013


CargoBot is an iPad puzzle game that teaches users basic programming logic through commanding a crane to move crates. The games includes 36 increasingly challenging levels that require users to control the crane via programming commands to move crates around. The game has several different levels of difficulty including an "impossible" level which will definitely inspire players to spend plenty of time trying to solve the puzzles.

Pros:

CargoBot does a great job of teaching the logic behind computer programming. The controls are easy to understand and use at the start of the game so there is a small learning curve. Students can record their solutions and share them on Youtube. These videos can be used as a way to get feedback on the userâ€™s process for finding the solution and as a way to show off their accomplishments in the game. After each challenge users receive a rating of one to three stars. Three stars are received if users move the crates in the least amount of moves.

Cons:

The game may be too challenging for younger students at first. New users should play through the tutorial levels before attempting any of the more difficult levels. These levels help users understand the game mechanics and relate them to actual computing logic.

Our Takeaway:

CargoBot can be the first step towards learning how to code. There are many digital resources to help users learn various languages (e.g., C++, Java, HTML) but very few that teach users the logic behind coding. CargoBot teaches logic like looping and conditional formatting. Once students better understand the logic they will be able to use this knowledge to more effectively learn and use different languages. The gaming aspect of the app makes learning code logic nearly effortless.

The Bottom Line:

CargoBot sets the foundation for learning to code.
