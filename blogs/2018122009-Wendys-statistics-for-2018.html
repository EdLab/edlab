Wendy's statistics for 2018

wendy

December 20, 2018

Hi.  I blog my statistics once a year on the last day before Christmas.  Everyday I keep statistics on how much I do and what I do.  I keep a spreadsheet of each thing and this way I only have to add up all the numbers.  Everything is together.  This is a culmination of all 12 months of the year.

Number of New Books Received- 449

Number of New Books Processed- 449

Number of Books Rush Processed- 73

Number of Books Searched for Doc Del- 583

Number of Books Searched for ILL- 47

Number of Books Brought to Reserve- 750

Number Of Cafe Items- 261

Number Of Books Brought to Hold Shelf- 186

Number of Missing Books Recovered- 484

Number of Books Shelved- 612

Number of Books Retrieved- 717

Number of Books Brought to Book Drop- 402


My favorite is finding missing books that are here.  The number of books received and processed is always the same.  The number of books rush processed signifies new books that go straight to reserve.
