Releasing this summer: Avengers 3d, Bright and shiny groups feature in Vialogues

Sharath Gururaj

April 30, 2012

Private please Hello folks, Have you ever wished you could group vialogues by some category and share it with a set of people at one go? Now you can! Introducing the new "groups" feature in Vialogues! We are planning to support the following use cases initially: 1. Creating a group. Create a group by giving a title, description for the group. You can add any number of your own vialogues to the group. You can also set moderators and participants for the group. Moderators of the group can edit the title and description, as well as add or remove other moderators and participants. A moderator can be removed from the group as long as every vialogue in the group has at least one of the vialogue's moderator as a moderator of the group. 2. Viewing and discussing a vialogue through group membership. A user can discuss a vialogue if at least one of the following hold: a. The vialogue is public b. The user is a direct participant(or moderator) in the vialogue c. The user is a participant(or moderator) of a group that contains that vialogue (new feature) 3. Viewing my groups Under the my profiles tab, there will be a new section called "My groups" where one can view his/her groups and delete or create a group There are other corner cases regarding permissions and vialogue access which I'll skip to keep this post simple. We feel this feature will enable a lot of new and interesting use cases 1. A professor can create a group of vialogues and share it with a class 2. Fun/Entertainment grouping like "My favorite cat videos" 3. Instructional grouping of videos like "Cooking 101" that any user can put together and share 4. Groups to educate on a broad content: "Machine learning" What other usecases/features do you think will be useful to users? Do you see any potential complications in existing use cases by introducing groups? We look forward to any feedback. Thanks!
