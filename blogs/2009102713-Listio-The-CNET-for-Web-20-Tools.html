Listio: The CNET for Web 2.0 Tools?

Ting Yuan

October 27, 2009

I happened to find this website this morning while doing the e-book research. Listio includes comprehensive web 2.0 tools in different categories, e.g., e-publishing, e-commerce, music tools, survey tools, visual search, etc. In addition to listing these tools in a timely manner, the website invites votes and reviews of tools within different categories. For example, there is a review of four e-book web publishing tools here. For next week's Development and Research meeting, maybe Listio can enlighten us to brainstorm about a website/application reviewing online educational tools/software.
