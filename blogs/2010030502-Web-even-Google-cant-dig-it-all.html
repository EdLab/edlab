Web - even Google can't dig it all !!!

Pranay Dharmale

March 5, 2010

In my Advanced Databases course, we recently discussed the hidden web which search engines can't grasp. This hidden data is beyond the regular web pages which get stored in the index maintained by the search engines. A search engine can index only web pages publically available to them but cannot access the restricted dataset. For instance, one case we discussed was: Query [wireless AND network] on USPTO (U.S. Patent) database:110,458 hits. Query [site:patft.uspto.gov wireless network] on Google:275 hits. It is not possible for Google to search USPTO site for every possible query to discover all the pages it might have. This opens up a whole new field, of extracting and classifying relevant information out of this enormous amount of hidden data, aiming to provide good (but "approximate") search results to users. Hence, the search results which we get are just an intelligent approximation out of all the relevant results out there.
