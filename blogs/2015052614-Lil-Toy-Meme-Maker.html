Lil Toy Meme Maker

tzaffi

May 26, 2015

** PRIVATE ** Hi Folks, In preparation for a developer share could you help me out by creating some memes as described below? The upcoming share is on Docker and I created a meme maker app for demonstration purposes. This uses an API that I found on the ultra-cool mashable API portal mashape. You can use it to make memes such as the following: I created this by navigating to the following URL:

http://ec2-52-6-217-231.compute-1.amazonaws.com/Crazy Dawg/no beer/for you

You can create a similar meme by composing a URL in the following way:
  1. Start with http://ec2-52-6-217-231.compute-1.amazonaws.com
  2. Add to it the name of the meme after a slash. In the case above the meme image is called "Crazy Dawg" so I added /Crazy Dawg to the URL. Here's a list of available meme image names.
  3. Add to it the top caption after a slash. In the case above I wanted "no beer" so I added /no beer
  4. Add to it the bottom caption after a slash. In the case above I wanted "for you" so I added /for you
  5. String steps (1) through (4) together to obtain a URL such as http://ec2-52-6-217-231.compute-1.amazonaws.com/Crazy Dawg/no beer/for you and pop it into a the URL bar of your favorite browser
I'd be curious what cool memes you all come up with. Just put a link to your new meme in the comments to this blog post. CHEERIO!!!!
