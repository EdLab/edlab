Microsoft Minecraft

Youssef Ballo

February 12, 2016

Microsoft Minecraft, a new exploration for students? Or a simpler way for teachers to teach? Microsoft released the education edition to Minecraft which already has its own features. The company currently wants to make it an educational product. However while reading this article, I became skeptical. When you think about the school system now and how it need some work and tweaks, you start to ponder whether this is basically an online curriculum. It is customized for school, however children don't develop the same learning abilities. There are children that visually learn, and certainly children that are more computer savvy than others. This is the era of technology but the biggest question is, "will it work?"
