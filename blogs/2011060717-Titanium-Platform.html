Titanium Platform

Alex Park

June 7, 2011

Today, Hui Soo told me to check out a relatively new development platform called Titanium Platform. It is something that sounds just too good to be true. If you maintain one codebase using your favorite web technologies like HTML, CSS, Javascript, Python, etc.: basically, the technology EdLab developers can use with both eyes closed. You can use that one codebase to create applications on the Desktop (Windows, Intel-based Mac, Linux), the iPhone, and the Android, with support for Blackberry promised for later. The best part: it is released on the Apache 2 license, so it's free for both personal and commercial use! Even if this Titanium Platform was really bad, who doesn't like free stuff?? ... Actually, it doesn't look half bad. I haven't gotten a chance to use it yet (I'm still trying to familiarize myself with the Vialogues codebase... I started yesterday), but examples of applications already developed on this platform online with this tool seem legitimately good. Sure, when it says it works on the "Desktop," it's not quite a full program like, say, Microsoft Word; it resembles more the "apps" that you get from the Chrome Web Store. However, for mobile development, which we are interested in, it seems really good! I'm sure there're downsides to using this platform over coding it natively, the apps on the iPhone seemed comparable even in usability to the apps on Android. When EdLab inevitably delves into mobile app development, especially since this uses technology so familiar to those already at EdLab, this seems like a great way to get things rolling. If I get a chance to use this platform to actually develop something, I'll post on the blog the results.
