I Am Woman: Watch Me Code

saraehardman

June 28, 2017


Just like superheroes, most coders are men. Yet with the fierce resurgence of Wonder Woman comes the perfect opportunity to inspire girls to try things typically discouraged by ridiculous gender norms. Made w/ Code capitalized on the new film to create a three-level coding game called Wonder Woman, centered on the mighty Diana Prince.

Pros:

Wonder Woman follows the same formatting of other Made w/ Code projects, where players snap together various actions and test the sequence to see if it completes their mission. In this game, players code sequences of movements for Diana as she leaps over obstacles, dodges swords, and lassos villains. If you code each sequence correctly, you win the round, get to watch a real clip from the movie, and advance to the next level.

Four coding aspects are taught in this trial-and-error learning experience: variables, sequences, loops, and conditionals. Players snap together blocks such as run 1 space and block with shield, then watch Diana carry out the sequence as she faces her opponents. The game provides hints when needed and allows an unlimited number of chances to snap the correct order into place.

Cons:

Wonder Woman is a project, not an entire platform or extensive narrative-based game. For this reason, there are only three levels in the game, in which only four coding skills are taught. Players will likely solve the puzzles within a matter of hours. But for a fuller gamut of coding skill practice, you can scan the 13 other projects on the Made w/ Code website.
