Wendy's statistics for 2015

wendy

December 17, 2015

These are my statistics for 2015. Number of New Books Received 1,078 Number of New Books Processed 1,078 Number of Cafe Items 684 Number of Books Searched for Doc Del 621 Number of Books Searched for ILL 235 Number of Books Brought Down from Range 8 116 Number of Books Brought to Book Drop 948 Number of Books Brought to Reserve 668 Number of Books Brought to Hold Shelf 735 Number of Books Withdrawn 282 Number of Books Rush Processed 168 Number of Missing Books Recovered 36 Number of YBP Books Processed 2 Number of Retrievals 453 Number of Books Reshelved 530 The cafe items number is the processed and withdrawn added together. Sometimes when I put a magazine out there the old one isn't there, so I can't count it. I did not count the math journals I still process and accept. These are a separate thing. The number of retrievals I add up only once a year. The reshelved books also is once a year totaled.
