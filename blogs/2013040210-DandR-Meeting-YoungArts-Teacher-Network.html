D&R Meeting: YoungArts Teacher Network

Janice Joo

April 2, 2013

Week 12: Zhou Zhou and George Nantwi You will need the following: Sign up for the YA MasterClass Teacher Blog here We haven't heard from them since last year at the D&R meetings, but the YoungArts team never really stopped. They've simply been in ninja-mode, because voila – the Teacher Network Blog is now ready for the teacher universe! This means that teachers can create their own lessons from the study guide, ultimately building a fantastic repository of teaching material for other teachers to share. The Teacher Network Blog will make its first public appearance in Las Vegas next week at the Sloan-C conference, so let's help the team test-drive the site and make it ready for some lights, camera, and action. Message From the Presenters In addition to going over the features of the Teacher Network, we've planned a brief activity in which you get to use the site and play around with some of its features. We've created a showcase Network site where you can find info on registering as an educator and/or student. Please leave all feedback here so we can work on improving the Network. Activity

  • Study the Showcase site (http://showcase.youngartsmasterclass.org/)
  • Then go to: http://edlabdr.youngartsmasterclass.org/
  • Each table signup an account A & C — teacher (signup in Teacher Network) B & D — student (signup in edlabdr microsite)
  • Try: Student: request access, respond to weekly quickpolls, make comments Teacher: approve access request, create quickpolls, view site stats, embed vialogues video
  • Feedback:
  • Request & approve access process
  • How to improve Showcase microsite
  • Promotion (YA, UFR, & future projects)
