Adobe Flex: A platform for quickly developing rich web applications.

fanwar

February 23, 2009

Recently, we at EdLab's Development & Research team have been looking at Adobe Flex and using that as a platform for developing an application that will (hopefully) have a rich front-end for the user while also being database driven and having a lot of business logic involved. Flex turns out to be a good framework when working on such applications. There is a good tutorial out there on ramping up to Flex. It might provide a quicker development option for those building media-heavy applications that also have a lot of backend logic involved. There are also flash tools available for mobile devices for those of you working on those apps.
