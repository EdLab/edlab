So Long, and Thanks for All the Fish

Dan Mallinger

April 9, 2007

Goodbye old computers! I'm impressed you made it this far but you're doomed. TSI has set up our Vista boxes to test drive the new operating system. So far so good. Vista has a number of new security features which take some getting used to, but may address the endless security holes prior versions of Windows suffered from. Obviously the interface is prettier, it even includes a great deal of transparency effects. Overall, Vista is much nicer to look at than XP. Another cool feature is the ability to store bootup data on an external USB device. If you choose to do this, you can have Vista bootup more quickly by inserting the device prior to bootup. But the really cutting edge features are Vista's new speech to text tools. You'll see these coming up in Meety, but hopefully we have some time to find other uses in the coming weeks. All in all, we're excited about the new machines; and as a thank you to the lab, we've assembled a few pictures of TSI putting the machines together. If you have some interest in Vista or would like to utilize some of the new technologies, let us know. Mi technology es su technology. Thanks again - TSI
