Language Learning Books Die A Slow Death

EdLab Studios

October 26, 2014


Language learning is becoming a core service of libraries, one on which patrons rely. If you can’t find the language you need on the shelf, chances are good your library subscribes to a number of online programs that’ll have you asking where the bathroom is in no time. However, if it’s a more personal learning experience you’re looking for, you’d best search elsewhere; just a small percentage of libraries offer in-person programming (like ESOL classes and bilingual speaking exchanges).

Source: Library Journal
