The Year of Code

Jenny Abamu

December 20, 2016


This year I took it upon myself to ride the programming wave and learn some code, and thanks to some talented developers, here are five memorable apps that I found useful:

1. Codesters
Making the integration of coding into classroom lessons as painless as possible puts Codesters at the top of my list.

2. Swift Playgrounds
With this game for learning Swift, Apple brings more of what educators want and need to the app store.

3. Pixel Press Floors
Introducing students to the concept of coding can be intimidating, but this app allows students to transfer their 2D drawings to the digital world.

4. Hackaball
Combining Bluetooth, code, and a ball is a recipe for hours of programming entertainment. Hackaball definitely delivers the fun, which is how it made my list.

5. Thimble
Remixing is moving from the music industry to computer science. Thimble allows students to learn programming through remixing developed code and putting their own spin on it.
