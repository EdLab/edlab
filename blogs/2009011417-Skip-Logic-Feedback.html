Skip Logic Feedback

HSC

January 14, 2009

I am posting this so that folks can provide feedback on the Survey Sidekick Skip Logic backend. Here is one observation I will share as an end user: The "Back" function is not working properly in the context of the skip logic. I initially answered "Yes" to the first question (Do you want to join the EdLab basketball league?) and when I pressed continue, I got the appropriate set of questions (What is the best day for you to play?). But then I went back to change my answer to "No." However, I got the same set of questions. So I logged off and logged back in. And even though I answered "No" to the first question, I still got the set of questions that should have been skipped. I look forward to seeing this feature being implemented in the Beta release. I will provide feedback on the backend/administration shortly.
