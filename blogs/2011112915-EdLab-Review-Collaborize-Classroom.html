EdLab Review: Collaborize Classroom

Fred Rossoff

November 29, 2011

Collaborize Classroom is another class-management program developed for secondary school teachers. It is designed to facilitate student discussion and collaboration through an online platform. Pros: Students in a class can be divided up into relevant groups, to allow for more structured and targeted discussions. The program offers a variety of tools to increase student engagement, including multimedia uploading, forums, and simple questionnaires. The Website also incorporates a resource library, where teachers can share and recommend activities they've developed for Collaborize. It also feels fairly user-friendly, which is important when seeking to encourage its use by reluctant students. Cons: Like many of these sites, the functionality is somewhat limited. The program can't serve as anything more than an adjunct to the real classroom work, and limits itself to providing an arena for short, focused discussions. EdLab Relevance: Collaborize is, in some ways, working in the same space as Vialogues. I wonder if we, like them, should start thinking about offering more teacher support (including model vialogues, tutorials, even group management features). In any event I think one thing the program does very well is demonstrate how useful content management can be, particularly when you're collecting content from your contributors. With the organization provided on the website, it's very easy to find relevant lessons. Our Takeaway: I'm not sure Collaborize has much to distinguish itself from the pack. It's possible that by limiting the program to specific discussions set up by the teacher, the creators are missing out on a fuller-use case, which could include putting more of the classroom experience online.
