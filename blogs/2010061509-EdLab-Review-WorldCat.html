EdLab Review: WorldCat

Chandler Kang

June 15, 2010

Edlab review: WorldCatÂ® Local Single search that connects people to all your library's materials http://www.oclc.org/us/en/worldcat/default.htm
WorldCat is a global network of library content and services that uses the Web to allow libraries in each institution be more connected, more open and more productive.
Price: Free How it works: WorldCat Local requires a WorldCat subscription to FirstSearch. It takes at least 24 hours to get received URL from WorldCat. With URL receive, institution can install and link it and its users can use it through the institution website. A user can search item by keyword, author, or title. For searching with title, you simply type title of the item in the blank form and it leads to a result page that shows the items from more than 9,000 libraries connected to each other by WorldCat. To get the most relevant result, you can use advanced search. Once you click the title link, it shows the item's detail record. To request the item, a user needs to login to prove that he/she is affiliated to that institution and, if he wants a physical book, he can type an address that he wants to get the item delivered.
Prop points: Basically, WorldCat is aiming at straightforward user interface based on user-driven design. WorldCat provides users with a single search box which is easy to access to materials. The search results are ranked by relevancy. The result also shows subject facets to help refine users' search. WorldCat searches cover art and provide the ability to tag, review and add items to lists. It can be linked to a local library catalog, Google Books and Amazon. In addition to these advantages, WorldCat help users save pains by providing a direct link to get the items. This means that users do not have to fill out another interlibrary loan request form, when it is not available in institution that the users are affiliated with. Finally, the usability keeps improving through monthly updates by OCLC.
Drop points: It is not yet a replacement for the local Online Public Access Catalogs (OPACs). Also, there are many library catalogs that are specialized on certain subject topics that are not fully covered by WorldCat.
Final score: WorldCat is a catalog of items owned by more than 9,000 libraries around the world. Users can request an item directly from the WorldCat record. It enables users to access materials easily and comfortably.
