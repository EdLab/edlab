Try out improved Survey Sidekick and give feedback

fanwar

December 19, 2008

A couple of us in the development & research team have been working on updating several features of Survey Sidekick and improving the user experience. We'd love to hear feedback from you about the tool and what features or changes you want to see to make it useful. There are two key goals for this tool to keep in mind: - We want it to be a useful resource for people who need to learn how to conduct survey research (doctoral students, perhaps?). - We want the tool to be useful as a research device beyond surveying - in particular, we would like it to be a survey tool that supports rigorous analysis of the results. We're still adding a few more feature before we release the beta, but we want to get as much feedback as possible for this release and beyond. You can visit the tool in development at http://dev.survey.tc-library.org/edit.php.
