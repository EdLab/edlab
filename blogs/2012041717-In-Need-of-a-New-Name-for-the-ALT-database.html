In Need of a New Name for the ALT database

Demetri Lales

April 17, 2012

PRIVATE Hey everyone, just wanted to give an update on the ALT product and ask for a favor. One of the main feedback we got from the ALT D&R was that a database that only included educational technologies that were fully adaptive wouldn't have much in it. So we decided to include more learning technologies that wouldn't necessarily be adaptive, which of course means that we can't call the project the Adaptive Learning Technologies Database any more. I was hoping to ask my fellow Edlabbers for a proper brand name for the Learning Technologies Database (LT DB?). What do you call a database full of different kinds of learning technologies, where each one would include a review where several professionals with different backgrounds (educator, researcher, developer, investor) come together to review and discuss a learning technology? Should we just call it the Learning Technology Database? Or can we give it better name? If one person comes up with a really good name I might consider giving him/her a special prize, maybe some spanakopita (spinach pie)?
