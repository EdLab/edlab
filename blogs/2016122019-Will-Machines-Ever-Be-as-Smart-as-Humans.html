Will Machines Ever Be as Smart as Humans?

saraehardman

December 20, 2016


What does it mean for a machine to be intelligent? There are many machines that can complete one task really well, such as playing AlphaGo or holding conversations. But for a computer to truly be considered intelligent, it must be able to keep up with human intelligence. And a human does not specialize in just one cognitive skill. Far surpassing the intelligence of one-track bots, humans can play AlphaGo, recognize faces, and have open-ended conversations, all at the same time.

It’s currently impossible to program the intelligence of a human into a machine. It’s not impossible, however, to program a machine to learn, and in doing so give it the potential to eventually achieve human-like intelligence. While human-level machine intelligence is still far from being realized, bots that use machine learning are improving quickly, as evidenced in social media sites and search engines.

The narrator in this vialogue explains that machine intelligence can be used to improve self-driving cars and internet searches. What are some other ways we could use machine intelligence? In what ways might machine learning have applications for human learning? Join the discussion on Vialogues.
