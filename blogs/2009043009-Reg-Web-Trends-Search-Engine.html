People might have already been aware about Google Trends, so I am going to shed some more light on other facets of how we can get Search Engine Data. Microsoft has been doing good work in this field allowing one to predict a website's user age, gender, demographic information, classification of website in a certain category depending on the contents of a website, tons of analysis related to search keywords like Google trends and it allows keyword clustering, keyword forecast and keywords expansion. It detects the commercial intent of visitors for any website.

Here is the link for the various tools provided by MSN for exploring all these options and also checkout the upcoming tools which will add a lot of value to the way we can use search engine data to make business decisions as more and more people have started using engines like Google, MSN, and Yahoo to go to the respective websites.
