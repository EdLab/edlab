Google's 'Magic Ring' Could Kill the Password

abagigah92

March 18, 2013

If you are a type of person that do not like using passwords, Google is currently researching a way to kill the use of the password. This will be done by using a trinket that plugs into USB slot on the computer, which will then authenticate the user. A principal engineer at Google, who works in security compared logging in to a site to be as easy as swiping a bank card. Overall, many people do not like the use of passwords simply because they have many different passwords for different sites. Additionally, Google view the use of a USB to be easier in authenticating passwords. Google's prototype USB uses piece of digital information known as cryptographic key. This is a software that serves to encode and decode methods of secret communication. The cryptographic key works such that it encodes a message that is unreadable to anyone but the intended recipient; it also allows us to read a code that is only meant for the user.
