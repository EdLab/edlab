The journey of investigating influential teaching and learning tools continues… Diver: Further Implication for Critter?

Diver is a video capturing, annotating, and sharing tool developed by the Stanford Center for Innovations in Learning in 2005. Similar to our suggestions for Critter, this software seems to have several advanced functions such as capturing a moment in a video, making comments, and releasing your “recreated” version on web. The current version is free but only available upon request. Please contact Ms. Mary Albertson via mary.albertson@stanford.edu if interested.

Etherpad: Multi-User Online Editing Tool

Etherpad.com is an easy-to-use textual collaboration tool for multiple users (teachers/students/co-workers). No sign up or log in is required. Since Google Doc cannot have many collaborators revise at the same time, this tool is a good choice.

Three steps:

Go to etherpad.com Click "Create New Pad" Record and bookmark URL of the created document to send to others and be able to return to your document to edit.
