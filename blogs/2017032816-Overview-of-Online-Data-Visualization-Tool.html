Overview of Online Data Visualization Tool

xk2120

March 28, 2017

There are a lot of free online visualization tools. It all depends on that type of data you have and what tasks you want to complete. Below are some of the best ones I’ve seen or used. 


Figure 1. Flowchart of data visualization tool choosing 

If you have the verbal data and want to do some text mining, Overview (overviewdocs.com) is a good place to start. You can make a word cloud and word co-occurrence graphs within 10 seconds. You can identify word frequencies and associated networks. (See Figure 2)


 Figure 2. Examples of Overview (Left: word cooccurrence; Right: word cloud)

If you are dealing with numeric data and want to quickly scan through the possible associations between variables, Google Data Studio (datastudio.google.com) is one of the easier tools to use to explore your data. You can simply create infographics by selecting the visualization methods and variables in the upper and left panels (See Figure 3). 

Figure 3. Operation Dashboard of Google Data Studio

If you are looking for a fancy data visualization, RawGraphs (http://rawgraphs.io/) creates beautiful graphs to serve your artistic sense of expression. Again, you just need to choose the display preference from the templates and map variables that you want to highlight ( See Figure 4). 


Figure 4. Operation Dashboard of RawGraphs

Plot.ly (https://plot.ly) is a more sophisticated visualization tools because it contains statistical functions such as ANOVA, T-test, and correlation. You can quickly test your hypothesis while trying to make decent data visualizations (See Figure 5).
