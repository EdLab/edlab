D & R: The New Data Dashboard

tzaffi

September 21, 2015

Hi Everyone! Neha and I are really excited to talk about the new Data Dashboard for today's D & R. You will be split up into four groups and are provided with a login and password for each group. With this information, you should log into the new Data Dashboard at: https://edlabddb.tc-library.org Questions for the groups

  1. Data Dashboard Interactive Exercise. For each of the following, find a Vialogue that fits the criteria and post a link to it on the blog. A. The most commented on Vialogue in July 2015. B. The most commented on Vialogue in July 2015 that was not from YouTube. C. Hui Soo's most commented on Vialogue ever. (See the D & R notebook for hints)
  2. Improvements. Based on your group, explore either the NLT Interactive notebook (Groups I and II) , or the Vialogues Interactive notebook (Groups III and IV), formulate questions and try to answer them with the notebook. Post answers to the following questions on the blog: A. What were the questions about NLT (or Vialogues) that you came up with? Were you able to answer them? B. Did you spot any usability issues that need to be fixed or improved? Please explain.
  3. Research. Post to the blog questions which you would like to be able to answer in Data Dashboard about one or more of the following: A. mSchool B. EdLab Marketing C. any other EdLab related statistics
