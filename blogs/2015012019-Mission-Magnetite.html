Mission Magnetite

Meredith Powers

January 20, 2015


Mission Magnetite is an interactive game to help kids understand part-whole relationships in math. It’s designed for grades 3-5, and the game is tied to national educational standards. Players must match up percentages, fractions, and visual models to defeat a rocket-launching villain.

Pros:

The game reinforces fractions and percentages mostly through repetition. Matching up the picture with the numerical values might help drive home the concept for more visual learners, while matching up the percentages and fractions drills the relationship between different representations of the same value. The game also includes more than one correct combination of values, so players learn to recognize that fractions can have the same value even if they initially appear quite different.

Cons:

The instructions are a little lacking in clarity. The picture version of the percentage is grey and white, and it’s not specified which color is meant to represent the part of the whole. Users will figure it out while playing, but it does add an initial layer of confusion to the game, and may even have some students second-guessing correct answers in the beginning.

Our Takeaway:

Since each round consists of a series of matching with no leftover tiles, it’s not hard to game the system by picking the easy matches first and saving the most difficult one for last—when it becomes the default match. As the levels move faster, it gets harder to pick and choose between easy and hard matches. The game provides a lot of repetition, so it works for drilling and practicing fractions and percentages. After awhile, the repetition starts to get dull. It might be more interesting if the levels increased in difficulty, rather than just speed. Still, the game clearly isn’t meant to hold up for extended play—just long enough to practice the concept.
