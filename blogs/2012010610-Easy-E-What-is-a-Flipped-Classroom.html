Easy E: What is a "Flipped Classroom?"

Kate Meersschaert

January 6, 2012

"The Flipped Classroom is an intentional shift of content which in turn helps move students back to the center of learning rather than the products of schooling." The Daily Riff, The Flipped Class Manifest, January 3, 2012 After exploring teammate Demetri Lales' many posts (like this one) on the potential for using Vialogues to flip the classroom and exploring leading "flipped" content providers like Khan Academy my best summation of this concept is that the main work of answering meaningful questions should be reserved for the classroom, while the work of learning basic concepts can be done at student's own pace using asynchronous technologies like Vialogues and YouTube videos at home and on learner's own time. This flipped model allows in-class time to be optimized for answering questions and helping students with what would have previously been considered "homework"... solving complex problems using the basic skills gained at home.
