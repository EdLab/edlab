From Trend Predicting To Pattern Mining

Sharon Hsiao

June 1, 2011

Google recently blogged about a new technology: Google Correlate. It's an advanced tool to mine patterns in searching data, and it is also considered an enhancement from Google Trends or Google Insights for Search. The original idea allows the user to enter a search term and expects the system to return a trend as the prediction (Google Trend). Now, the new tweak on the Google Correlate tool is an implementation of the reversed technology: users get to enter the trend of some real world activities. Utilize the data points to fit the line and see which search terms best match the trend. What's so exciting about the tool is that we no longer have to carefully choose the query terms (having some pre-knowledge) in order to find the phenomenon. We could now pour a bunch of data by giving a temporal or spatial pattern of interest, let the technology determine which queries best mimic the data. What's the implication of all this in today's educational environment? We are all aware of the complexities and abundance of information that profoundly affects us in the modern e-learning environment. That's where Learning Analytics plays a very important role in this dynamic learning environment. (For more information on Learning Analytics, refer here). Pattern mining and trend predicting are two key elements in Learning Analytics that complement each other. The new tool today, Google Correlate, opens up possibilities to act on the data to spot potential issues or predict learners' future performance. We can now also expect how these tools restructure the process of teaching and learning, such as a bottom-up Learning Analytics process instead of traditional top-down approach, and lead to the road of lifelong learning.
