Gamestar Mechanic

Demetri Lales

August 28, 2013


Gamestar Mechanic is a platform that provides a framework for creating online games. It also allows users to publish their games and share them with other users. Gamestar Mechanic has an active community of game designers who provide reviews and feedback on ways to improve each game. When users first sign up for the site they have the option of playing through five game quests. Playing through these initial games unlocks sprites that users can employ when they create their games. The site also provides lesson samples for teachers to integrate in their curriculums. Gamestar Mechanic teaches users 21st century skills, systems thinking, and creative problem solving.

Pros:

The platform encourages users to flex their creative muscles and create their own games while learning useful skills for modern jobs including design, collaboration, and programming logic. The process of creating games uses basic programming logic which makes the platform a stepping stone to more advanced development. Gamestar Mechanic includes a few resources for teachers interested in using game-based learning in their classrooms.

Cons:

When users first sign up for the platform, they have a limited supply of tools to customize their games. To unlock more tools and sprites for customization, users need to play through the games in each quest. This step helps familiarize new users with the types of games they can create on the platform and gives users ideas and inspirations for creating their own games.

Our Takeaway:

Gamestar Mechanics is a learning platform that teaches users skills like design, collaboration, and programming for 21st century jobs and STEM careers. Building games requires students to think systematically and solve problems during game creation. These skills are similar to the way programmers write code. The platform could serve as a prerequisite for young students who might develop an interest in programming. Students who want to learn more about programming to create games can move on to more sophisticated game development platforms like Scratch.

The Bottom Line:

Gamestar Mechanic lets you play and create 2-D games.
