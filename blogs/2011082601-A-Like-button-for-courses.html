A Like button for courses

Ankit Ranka

August 26, 2011

Today while discussing the design for the favorites functionality in Pundit I had an 'aha' moment that I want to share. As the title explains, how about having a favorites button for courses, similar to Facebook like. All the departments at TC will be able to embed that button on individual course listings. We already have individual page for each course, so total favorites could be shown there. I think the first place students look for courses is the department website and if we have a favorite button there, it will increase the visibility of Pundit as a resource and also in future we might have a 'similar course' functionality. As soon as you favorite a course, Pundit will show you similar courses offered in TC, Columbia or Online. Another use can be that if we don't have a course that user favorites we will be able to crate a course page on the fly that a user will be able to add information to. So this will be more like suggest a course but this time Pundit fulfilling users wish in real time. So imagine I favorite the 'iPhone programming' course on example.com and that results in a course being created on pundit based on the URL of that website, this is what happens when you hit facebook like button on any website. And then user will be able to add more detail of that course on Pundit. I haven't given much thought to the idea but is this something that will be useful for students? What do you guys think?
