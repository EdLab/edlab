Thimble

Sarah van den Berg

January 11, 2016


Thimble is a product of the Mozilla Learning Network that makes it easy to learn and teach coding. Users can start a project from scratch, remix a project, and download a teaching toolkit to lead a class through a project.

Pros:

The projects are engaging and directions are clear. Students can remix a Homework Excuses Generator, a Keep Calm and Carry On poster, and a Back-to-School Postcard. Students dive immediately into the code to learn the basics of CSS, HTML, and JavaScript. Light gray comments in the code explain what the lines mean and how to make changes. On the right side is a split screen layout, where the user can toggle between "preview" and "tutorial" view to reveal additional challenges like playing with the stylesheet and adding font families from Google. Even teachers with no coding experience can use the clear plans in the teacher toolkit to guide themselves, and then their class, through the remixes.

Cons:

It took about five minutes as an adult with practically no coding experience to remix "My Six-Word Summer" into "My Six-Word Thimble," with new banner sizes, background images, and text. Students, with even a bit of coding experience, may work just as quickly through the tutorial. Although students can create projects from scratch, there are only six remix projects. Teachers can make their own tutorials, so this might be an ideal platform for a computer science class, led by someone with the expertise and time to do that.

Our Takeaway:

We envision Thimble working across a spectrum of abilities. As a beginner tool, it is a great starting point for a class learning to code or developing web literacy. What are drawbacks for beginners (for example, limited projects) might be benefits for more advanced students (a clean, easy to use platform to create individual work). Embodying the ethos of the web, Thimble is a platform for building on the work of others, making it yours, and giving back to the coding community.
