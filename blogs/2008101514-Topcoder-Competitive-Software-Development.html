Topcoder - Competitive Software Development

fanwar

October 15, 2008

For those of you looking for the next programming challenge after Scratch, I recommend you check out Topcoder (http://www.topcoder.com/). It is a site that hosts competitions and challenges for all phases of software development (from architecting an entire software product to creating new algorithms for specific tasks and everything in between). The nice feature about this site is that it is set up as a competitive environment where users play against each other to develop the best piece of software. This approach has even attracted high schoolers to the site. Computer science teachers can use the competitions on Topcoder to reinforce or challenge student learning by having them participate in competitions. Even ESPN has gotten in on the act: it has teamed with Topcoder to create a sports-themed programming contest (http://winningformula.espn.com/)
