Frank Webster

April 13, 2006

SJCPL Subject Guides

http://www.libraryforlife.org/subjectguides/index.php/Main_Page

"St. Joseph County Public Library does its web guides as wikis. While only librarians can actually change content, each page welcomes comments from library users and requests for new topics to be added. An interesting model."
