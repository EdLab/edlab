The Public Art Project That Will Never Be Finished

Rebecca.Sullivan

September 10, 2018


Abilene Christian University(ACU)’s Lightwalk project shows how everything on a campus, even public art, can be part of the curriculum. The interactive light display integrates hardware and software to liven up the quad, but it also invites students, now and in the future, to make their mark by adjusting and expanding the display.

By designing new effect codes, students can use the Lightwalk as a space to experiment with public-facing design. It’s also a place to hang out after class. Do you think public displays like ACU’s Lightwalk can enhance learning? Add your voice to the discussion on Vialogues.

Excerpts from the discussion:

@00:54 Dallas Milanovich: I have a park near my house I like walking my dog through in the evening, but what I like about it is that things are always changing - nature is never static. Artistic and tech displays that emulate this interest me, so I think I'd like this Light Walk if it's constantly changing.

@01:38 Sara Hardman: Interaction enhances experience, and I think the same is true with artistic displays. If people can help shape them, they'll be more interested not only in how they look, but in how they work as well. I think giving students this opportunity will make it a better learning and aesthetic project.
