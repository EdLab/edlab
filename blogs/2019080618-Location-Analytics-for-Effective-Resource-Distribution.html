Location Analytics for Effective Resource Distribution

chingfu

August 6, 2019

New York City’s subway system has been struggling to improve its overcrowding issue and on-time performance. One of MTA’s latest efforts is to adopt location analytics technology to get real-time passenger locational information for its operation decisions. This solution is provided by Veovo, one of the six finalists that won a competition by MTA’s tech accelerator program Transit Tech Lab to address these problems.


Veovo's proposal uses a combination of various sensor and deep learning algorithms to provide a real-time overview of passenger volumes. It aims to detect and predict irregular subway operations so that MTA can better deploy its staff and resources. It would also help users make better travel decisions as more NYC subway information becomes available (platform crowdedness, delays, constructions, etc.)


Location analytics can be useful for learning theater resources planning and configuration. Malik and I are recently working on an exploratory project that uses Quuppa to track how various furniture pieces (such as whiteboard walls) are deployed and used overtime in various learning events. The idea is to gain insight about furniture and other instructional resources usage at the theater, and to demonstrate the flexibility of learning space configuration at the theater. Let us know if you have ideas about location analytics use cases for the learning theater. Check out a previous case study by Hui Soo, Geoge, Gary and me.
