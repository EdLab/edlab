Looking for a small group to learn a new language! Processing!

Janice Joo

June 6, 2012

Private: Hello, This is just a general call to the EdLab group to see if anyone is interested in figuring out the Processing language! My goal: To find a group of few people who want to run mini-experiments, and then share the code/process with each other so that we can eventually be equipped to come up with our own custom data visualizations for NLT. Processing can be converted to JS (rather simply, from what I understand) for online viewing capability via Processing.JS. One step at a time though... If interested, let's get started somehow... Note: I have a data vis D&R meeting scheduled for June 26th... we can show some demos then... if we get that far... It's my goal to personally have something for then though... And also... karaoke night? We all want to hear Jo sing... Janice
