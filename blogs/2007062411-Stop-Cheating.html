Stop Cheating

HSC

June 24, 2007

Cambridge, Mass.-based Software Secure has released a new device that promises to stop cheating in online environments. Called Remote Proctor, a small Web cam (which points into a reflective ball thereby allowing it to capture a full 360-degree image) and microphone is set up where a student takes the exam (see picture here). Once the test begins, the device records audio/video (I know it sounds like Meety), and software detects "disruptions" in the recording (a phone call, leaving the room). Based on the recording, an instructor can examine portions flagged by the software to determine if cheating has occurred. Although the product seems like a good way to prevent cheating, it raises all sorts of questions about privacy, assessment, and learning:

- How many of you would use the product as an instructor?

- If you were a student, would you enroll in a program that made use of this technology?

- What might you do as a student to sabotage this monitoring system? Read more about the use of Remote Proctor at Troy University in Alabama.
