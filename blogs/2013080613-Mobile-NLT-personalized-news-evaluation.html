Mobile NLT personalized news evaluation

Sharon Hsiao

August 6, 2013

* keep PRIVATE please Motivation: multi-touch interaction data has been found to increase web search result relevance. It hasn't been used in collaborative filtering recommendation. NLT provides us abundant content and platform to pioneer a new recommendation mechanism. Goal: 1) Test mobile NLT reading behavior. 2) The collected data will be studied for generating better recommendation in the future. Task: - Please follow this link to access selected NLT articles from you mobile device, (link will be provided through email individually) - Explore and read articles - Bookmark at least 10 articles(bookmark those interest you) before this Friday (2013/08/09). Ps. the current testing environment is ONLY an experimental mobile web and optimized to iPhones, iPods, iPads. If you are an android user, please use firefox, chrome or safari browser to complete the task. The UI/UX is NOT our concern, the focus is on the content. Thank you very much. reference: Mining touch interaction data on mobile devices to predict web search result relevance (2013) Qi Guo, Haojian Jin, Dmitry Lagun, Shuai Yuan, Eugene Agichtein
