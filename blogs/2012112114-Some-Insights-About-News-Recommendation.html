Some Insights About News Recommendation

Sharon Hsiao

November 21, 2012

Inspired by a recent New Learning Times (NLT) meeting, I want to share a couple of studies and results with you about personalized news. Since we want to provide personalized educational news down the road, I thought it might be useful to learn from previous experiences of this model. 1. Personal user interest profiles are more important for the news article recommendation process than public trends. Qi Gao, Fabian Abel, Geert-Jan Houben, Ke Tao (2011) Interweaving Trend and User Modeling for Personalized News Recommendation, published in Web Intelligence and Intelligent Agent Technology (WI-IAT), 2011 IEEE/WIC/ACM International Conference 2. Utilize click behavior for news recommendation as users are more likely to be influenced by local news trend. As such, a combination of user model for recommendation could include genuine interests based on past click history and current interests (clicks). This can be demonstrated to improve the quality of news recommendations and attract more frequent visits. Jahui Liu, Peter Dolan, Elin RÃ¸nby Pedersen (2010) Personalized News Recommendation Based on Click Behavior, Proceedings of the 15th international conference on Intelligent user interfaces 3. Users prefer transparency and control in their systems, and generate more trust to the recommenders. (That's the general effect of open user modeling. If you want to learn more, please refer to my thesis section 2.1.2 page 30 here). In news recommendation, such ability to edit user profiles may also harm the system performance. (Though Google News is doing it now). Ahn J.W., Brusilovsky, B., Brady, J., He, D. & Syn S.(2007) Open user profiles for adaptive news systems: help or harm?, Proceeding WWW '07 Proceedings of the 16th international conference on World Wide Web 4. Given a hierarchy of newly-published news articles, news groups that the user might prefer are differentiated using the long-term profile, and then in each selected news group, a list of news items are chosen based on the short-term user profile. Lei Li, Li Zheng, Tao Li (2011) LOGO: a long-short user interest integration in personalized news recommendation, In RecSys '11 Proceedings of the fifth ACM conference on Recommender systems There are a couple more but those are more algorithmic and specifically deal with improving the recommendation precisions. I'd love to talk about those, but i'm afraid I'll bore you with formulas. So, lets summarize these couple studies. It looks pretty clear that, user profiles (users' interests), is key. However, the recommendation algorithm should be able to detect long and short term interests. And sure, if the server and algorithm is powerful enough, it will allow users to easily "modify" their interests at anytime with no harm to the recommendation.
