TC Network resources at home

Anthony Cocciolo

June 3, 2006

If you have ever been interested in connecting to TC Network resources from home, such as Exchange through Outlook, servers (TCR, Rhapsody, Rhythm), Printers, File Shares, etc., it turns-out it is incredibly easy.

For Windows XP users, go to:
Start -> Control Panel -> Network Connections -> Create a new connection -> Connect to the network at my workplace -> Virtual Private Network -> enter Teachers College for company name -> Do Not dial connection -> and enter adagio.tc.columbia.edu for the server name -> do not use smart card -> Me only -> use your exchange username and password to login.

Once the connection has been made, you can connect to any resource like you were on campus.  For example, you can go Start -> Run -> \\tcr\public

Sorry Mac users, this won't work for you.
