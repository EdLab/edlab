Microsoft Surface and its Potential for STEM Education

Demetri Lales

January 31, 2012

The Microsoft Surface is a relatively new device that is a surface computing platform which responds to touch, either from hand or an object. Multiple users can interact with the Surface simultaneously and thus allowing for collaboration. It can also respond to everyday items like a paintbrush by using cameras for input. The Surface can also sense the pressure of someone's or something's touch, opening new possibilities for apps developers. Potential for sySTEMic Change: Now that we have more information on the Microsoft Surface let's think about how this device can be used for STEM education. From the features explained above the Surface does bring something that is very vital to all of STEM education, the capability to collaborate. Several students can surround the table and work on a project together using the Surface's applications. It is very hard to imagine multiple students working on one regular tablet together, so device can be used to encourage collaboration, which is vital in all STEM fields. Although the Surface does not have too many apps now (here's the list of new apps it does have), the device itself can provide educational app developers the chance to make some very engaging and collaborative applications. One idea for an app could be a simulation for dissecting frogs or any other organism where students can use actual tools with the Surface. There can be much more simulation type of applications that can be implemented using the Surface, which can help students understand what the STEM fields are, and how they fit in the real-world. How do you think the Surface can be used in STEM classrooms? Let's starting thinking about how we can use tools, like the Surface, to improve the education of STEM subject, everyone.
