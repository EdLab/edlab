Code Brooklyn

Jenny Shen

May 15, 2016


"Our mission is simple: every Brooklyn public school student gets the opportunity to learn to code at school. We will achieve the goal in the course of the next seven years." (Code Brooklyn)

Every Single Kid in Brooklyn

Itâ€™s no secret that a nation-wide push to get students coding at an early age is underway in schools through edtech entrepreneurship programs. Code Brooklyn is an example of what it looks like to create school- and city-wide access to programming classes. In fact, the organization is so ambitious, that it wants to get every student in Brooklyn coding within seven years. So while not every school is properly equipped with tech resources, Code Brooklyn provides tools for students without computers. "My Robotic Friends" and "Binary Baubles" are projects that use materials like plastic cups and paper printouts, making coding learnable even without having devices in the classroom.

The available curriculum options that Code Brooklyn features on its website are presented by TEALS (Technology Education and Literacy in Schools), which was started by a Microsoft employee seven years ago and pairs up professionals in the industry with educators to teach computer science to kids across the country. The "Tutorial Picker" is tailored based on the types of devices available, the age range of the students, access to fast internet, and even whether there are accommodations for blind students.
