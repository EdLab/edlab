Gooru

Demetri Lales

December 2, 2013


Gooru provides teachers with a library of online learning resources curated by their fellow teachers. Teachers can use this library to create their own courses and assign them to students. The site encourages users to organize videos and other media by tagging them with metadata. These tags make it easy for teachers to find the right resources by school level, subject, or desired media type. Once teachers find resources, they can group them into a single collection and add their own multiple choice questions. These collections can be assigned to students by sharing a classpage link to learning resources and assessments.

Pros:

Teachers have the option to pull learning resources from Gooru’s library or add their own for greater customization in their course. Searching for specific types of lessons is easy, because teachers can filter by format and the library includes videos, interactives, questions, slides, textbooks, lessons, and exams. Users adding resources are encouraged to categorize new submissions by grade level, subject type, and Common Core standard. A unique class code is generated for every class page. Teacher can give this code to students to grant them access to a class page and ensure privacy.

Cons:

Managing class pages and creating assignments may be a bit confusing for some. In order to assign learning resources to students teachers need to first gather materials into a collection and then add them to an assignment. It could be beneficial to allow teachers to add learning resources straight to an assignment folder. This would make assigning resources to students as easy as finding resources.

Our Takeaway:

Gooru provides a community of learning materials and space for online learning. The platform’s filter system makes it easy for users to find specific learning resources. Users also have the option to add their own learning resources, which may be an attractive option for teachers. The resources added by users contribute to the growing library and are available for others to find and use. This teacher curated library of digital learning resources should grow over time.

The Bottom Line:

Gooru gives teachers the space and resources needed for engaging online learning.
