Write Your First Angry Bird Program

chingfu

December 10, 2013

Last weekend I got a chance to learn An Hour of Code (also see more at NLT High Five) and found Write Your First Computer Program tutorial a well-designed introductory online resource for kids as young as age six to learn about basic concepts of coding and computer science. Here are some quick observations of this tutorial:

  • It uses Blockly, a drag-and-drop-block visual coding language that works intuitively for most beginners (Similar to those popular programming language such as Scratch and App Inventor for kids and youth)
  • The popular Angry Bird and its enemy green pigs are the main characters in this tutorial program
  • The tutorial includes videos of celebrities like Mark Zuckerberg, Bill Gates, Chris Bosh talking about the value of learning programming and introducing basic programming concepts such as loop and "if" statement in-between twenty learning tasks.
  • Learners can choose to skip videos and read text summaries of major ideas from the videos
  • "Show Code" option allows learners to see their programming blocks represented in JavaScript form
    • Overall, I think Your First Computer Program tutorial is a great one-hour self-directed learning resource for kids who would like to get their first taste of programming. There are several other hour-long online programming courses that all look engaging for beginners.
