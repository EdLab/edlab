Javascript Help

Jeannie Crowley

January 13, 2010

Hello All, The Check Republic team is a contender in the Firefox Jetpack for Learning Design Challenge. We've crossed some major hurdles, but could use a little help with the next step: passing variables from our Jetpack extension to our main URL. If anyone knows how to use javascript to get the variables from an extension to a URL displayed in the slidebar, we'd appreciate the help. (Or, if you just want to find out what the heck a Jetpack is. . .stop by!)
We need to pass $url, $urlTitle and $selectionText a php page. Our code so far:
//import future features that are not fully developed jetpack.future.import("slideBar"); jetpack.future.import("selection"); //add and edit slidebar jetpack.slideBar.append({ //load slidebar URL and icon icon: "http://160.39.76.162/jetpack/images/CRicon.ico", url: "http://160.39.76.162/jetpack/login.php", width: 350, }); // When a tab loads alert the page title and URL jetpack.tabs.onReady( function(urlinfo){ //create variables for url and url title var url = jetpack.tabs.focused.contentWindow.location.href; var urlTitle = jetpack.tabs.focused.contentWindow.document.title; console.log(url); jetpack.tabs.focused.contentWindow.alert('The URL is '+ url + ' The page title is "'+ urlTitle); }); // Highlight the selected text //jetpack.selection.onSelection(function colorer() { jetpack.selection.html = ""+ jetpack.selection.html + ""; //Create a variable for the selected text and send alert of selected text console.log("[jetpack] TEST"); var selectedText= jetpack.selection.text; console.log(selectedText); jetpack.tabs.focused.contentWindow.alert('The selected text is '+ selectedText); });
