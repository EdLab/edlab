Today's D&R: Bootstrap 101

Nara Kasbergen

October 22, 2013

Today's D&R will be an interactive workshop similar to the Twitter API D&R I gave last spring. As preparation, you have two ways to participate in the workshop: OPTION A 1) Please download & extract this zip file and have the folder with the contents open & ready when the D&R begins. 2) Unless you already have a favorite HTML editor, please download & install Sublime Text: http://www.sublimetext.com/ 3) Right-click index.html inside the extracted project folder and go to Open With > Sublime Text OPTION B Using Plunker, a web-based code editor with live preview: http://plnkr.co/edit/UQN3Qi?p=preview Please pick which method you want to use and familiarize yourself with the UI for a few minutes before the meeting so we can dive right in from the start. Also, you really won't need to know much about coding for this workshop, but if you feel very uncomfortable with basic HTML, please plan to buddy up with someone. We'll also make sure there's a developer at every table to assist. Here are the five tasks I would like you to complete: Task 1: Edit some text BEGINNER Go to CatIpsum.com or your favorite Ipsum generator. Replace the text on at least 1 page. ADVANCED Add a new page (tab) in both the content and navbar. Add a blockquote, a list, and/or a table with data. Task 2: Edit the form BEGINNER Add a new form element (eg. radio button, select box) with a label and proper Bootstrap syntax/formatting ADVANCED Change the form layout to horizontal Task 3: Edit the gallery BEGINNER Add 1 or 2 additional images to the carousel ADVANCED Add captions to the images in the carousel Add other kinds of navigation buttons in the carousel Task 4: Edit the navbar BEGINNER Switch to nav-pills or tabs. Re-order/re-align the links. ADVANCED Get rid of the tabbed content entirely. Make one continuous scrolling page and add a Scrollspy. Task 5: BONUS! If you've actually made it this farâ€¦ 1.
