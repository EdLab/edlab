Voice Response Translators — Can you hear me in your language?

Linda Bloom

December 17, 2006

I have always thought that an automatic voice activated translator would be a great tool to communicate with other languages. These types of translation applications have been around for several years, but for the most part little development has been made to expand their abilities –this is changing.

The military is currently using a hands free system that allows the user to continue daily activities while speaking through a Voice Response Translator. The company Integrated Wave Technologies has developed the MilTrans system that can translate English into 125 languages and then broadcast the language of choice. The system is designed for high noise areas so usability is probably conducive to several types of environments.

What is interesting about this type of application is how it may be used for learning and information purposes. When working and/or teaching in multicultural situations, systems such as this may alleviate some of the challenges for non-native communications.

Areas of eLearning on a global level can benefit so that non-native speakers may be able to participate equally in discussions and overall class projects; similar to a type of social acceptance and comfort when hearing communications in a native language.

This reminds me of how diverse cultures have used visual communications to generate a common level of information understanding via images. I can see that the technology of language translation also has the potential of evolving into a socially acceptable way to communicate. So that audio and visual real-time applications can be conducted in real-time self-lingual learning.

It will be interesting to see if this can be promoted towards global learning imitatives.
