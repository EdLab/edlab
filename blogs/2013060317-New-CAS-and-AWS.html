New CAS & AWS

Josh Lopez

June 3, 2013

The process of moving our sites to cloud hosting has just taken a big step forward. ResearchBroker is now officially CASified. Yudan's new CAS system is set up and we have created some simple documentation to help everyone else get the rest of our sites on the new CAS system. NLT is already connected to the new CAS so we're almost half way there. We'll be meeting with the other developers sometime in the next day or two in order to set up CAS on EdNode, SurveySidekick, & Vialogues (all of which have working, or at least partially working AWS instances). Finishing the AWS instances is probably going to take some time, but I've already set up several EC2 servers, and I have some documentation I created if anyone needs any additional assistance. After that, it's just a matter of connecting the DD APIs, which are already almost completely finished, and we're ready to make The Switch. 9 Days and counting...
