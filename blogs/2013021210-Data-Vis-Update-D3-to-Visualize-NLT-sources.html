Data Vis Update: D3 to Visualize NLT sources

Janice Joo

February 12, 2013

Hello Team, Since attending Dr. Hsiao's data visualization class last Tuesday, I've been exploring the D3 (Data Driven Model) Javascript Library with the hopes of creating interactive data visualizations compliant to web standards (moving away from the alternative, which is Flash). Here is one of the ideas I've been working on. The objective of this visualization is to:

  • Visualize the diversity of High Five sources and give the user the opportunity to explore High Five articles by source (and not just by date published)
  • Give the user the opportunity to visually track the growth of each stream over time
Here is my attempt so far. Using the D3 library, I can draw SVG elements, bind data to them, and give them fixed positions using simple math. But here is the technique that I'd love to apply: Apply their force layout! And here is my quick sketch for what I want to accomplish: Note, upon hovering/clicking a bubble, I want to populate the callout box
with HTML hyperlinks to the respective NLT articles (that data is not in the CSV right now...) Anyways, I just wanted to test it out because the backend is not clean for this kind of interaction. We'd need to commit some time to that, but before we make any big proposals, I'd love to get your feedback... Once the functionality is figured out, I can use CSS to style this up all pretty because D3 simply uses DOM elements!
