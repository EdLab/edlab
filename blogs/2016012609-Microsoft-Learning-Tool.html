Microsoft Learning Tool

Youssef Ballo

January 26, 2016

Microsoft Learning tool is a software to help students with learning challenges. The software adds a collection of tools to OneNote and can be used as a game and learning engine. It is also a tool for quizzing. This technology is used to make reading and writing more available to students. Researchers have acknowledged for years that reading happens faster and with better understanding when the text is presented in different visual styles. The learning tools take readers on an important step beyond the flexibility of eBooks, because you can use the software to read anything. You can copy text in a note and also use Office Lens app to photograph a page of the book and the OCR engine will convert it into text that the learning tool can manipulate. This software can be used in classrooms because if you have students with learning challenges, this tool can help them read and also teach them grammar.
