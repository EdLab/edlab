Critter 0.51 Released - critter.tc-library.org

fanwar

May 3, 2010

** Please keep private Please check out the latest version of Critter: critter.tc-library.org Some new features for this version: - Nicer URL - Automated signup (don't have to email requests anymore) - Ability to reply to comments - Backend and UI improvements such as nicer access control UI We'll have more versions coming every few weeks. Here are some things we're working on: - statistics on commenting activity and indexing of comments to support video search. - integration with social networks such as posting comments to facebook and linking to the specific point in the video you are talking about. - Polls and surveys embedded within video discussion - Other UI and backend improvements (better access control lists, distributed streaming of content for performance, etc.). Your feedback has been invaluable - here is some reply to common questions: Some have posted concerns about the security warning that sometimes comes up when logging in - we are working to resolve that (we need to finish installation of SSL certificates for the site). Others have suggested YouTube integration - this is not in our plans right now because of complications with data ownership issues and accessing YouTube's video player API. We will consider this as a feature add in the summer. Please keep the feedback coming and let me know if you want to do a pilot use of Critter with some group outside of Edlab and we can coordinate. We are still in internal release mode (with exceptions made on a case-by-case basis), but hope to release more widely in the coming weeks.
