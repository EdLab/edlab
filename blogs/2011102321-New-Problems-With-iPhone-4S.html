New Problems With iPhone 4S

Daniel Vergara

October 23, 2011

When I was searching for a new phone (currently own a BlackBerry), I was seriously considering the new iPhone 4S. I came across this article that highlighted the five major flaws of the new iPhone. The article automatically caught my attention because one would think that since this is the fourth installation of the phone, there would be few, if any, problems associated with it. When the 4S was first introduced a few weeks ago, Apple said it would be much faster than previous versions of the iPhone but there are still complaints of slow services, especially for Sprint users. Sprint is the latest service to carry the iPhone. Other problems associated with the iPhone 4S include the screen, the much talked about Siri voice assistance and camera. As the EdLab continues to develop new and more cutting edge technology to support education and for the general public, I wonder if we can learn from Apple and make sure that when we roll out any advanced features on new and existing products, we don't run into the same problems as Apple and other companies.
