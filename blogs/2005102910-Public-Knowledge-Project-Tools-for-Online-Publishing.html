Public Knowledge Project Tools for Online Publishing

gnatriello

October 29, 2005

As a follow-up thought to Nabeel's presentation on the new TESOL online journal site using zoops, I want to call our attention to the tools being offered by the Public Knowledge Project from the University of British Columbia. I was on a panel at AERA with John Willinsky who heads this open source project, and his group has clearly put a lot of thought into developing these open source tools. There are both tools for publishing a journal and tools for operating an editorial office. AERA has adopted the latter to run the editorial offices of all AERA journals. The TSI group might want to consider some of these tools as part of our light online publishing solution.
