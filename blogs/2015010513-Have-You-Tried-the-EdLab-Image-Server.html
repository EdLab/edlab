Have You Tried the EdLab Image Server?

tzaffi

January 5, 2015

***PRIVATE*** Have you ever needed a quick and easy way to put images up on the web? Did you know that Pranav created an in-house solution for this problem a couple of years ago? If you didn't know about this, I just wanted to re-publicize this helpful tool. I found it to be extremely useful today as I was writing my blog post about Node.js web servers and found myself running out of space for images on the EdLab website. You just go to our image server and drag the images you want to use as pictured here: To use the site just:

  1. Head on over to https://imageserver.tc-library.org/
  2. Choose the application that you want to use the images for (e.g. "Other")
  3. Drag and drop your images from your computer drive to the image server (as pictured above)
  4. Click on the Start button to commence the download
  5. You'll be provided with a url which you can just copy and paste inside your img tag's src attribute
