mSchool Ideas: D&R Meeting April 17, 2012

HSC

April 17, 2012

* KEEP PRIVATE - Brainstorm a list of mSchool offerings/educational experiences. - For each offering, describe the target audience (e.g., background, size). - What form would each offering take? - What tools, if any, do we need to create in order to support these educational experiences? * Keep in mind the EdLab mission and the goals/objectives of the mSchool (a virtuous cycle where educators, learners, developers, and researchers are collaborating in "EdLab" mode; a way for EdLab to study learning processes and applications, etc) as you discuss these questions. -- If you have time, how would you promote each offering?
