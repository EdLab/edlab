MIT's Popular Programming Language Scratch- Blocked!

Adrienne Garber

May 4, 2010

Alpha test version of BYOB (Build Your Own Blocks), an extension the popular visual programming language Scratch, is now available for download. BYOB adds first class lists and procedures to BYOB's original contribution of custom blocks and recursion. Scratch was developed by the Lifelong Kindergarten Group at the MIT Media Lab to make programming languages more accessible to young people. Using Scratch, students construct programs by snapping together primitive blocks that control multimedia multi-character presentations called sprites. According to many educators, Scratch allows kids to create interactive stories, animations, games, and more, while learning about mathematical and computational ideas. As an example, see below video from the 2009 Teachers College Scratch Day. BYOB also supports object oriented programming (OOP), either by using sprites or building the OOP facilities explicitly.
