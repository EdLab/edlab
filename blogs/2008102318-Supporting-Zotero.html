Supporting Zotero ?!?

Anthony Cocciolo

October 23, 2008

When I was in a meeting with Academic Computing this morning, we realized that the Endnote version that the Columbia Libraries provides doesn't work on the PC (they are missing the product code to actually make the program load). This will be resolved by AcIS. But it did raise the question if Zotero is the better citation management system (and why should anyone use Endnote anymore?). It seems to me pretty darned good. When you are on something that is citeable in Firefox, it shows an icon that you can use to drag into your citation repository. You can then create a biblopgraphy from your repository using whatever citation format (APA, MLA, Chicago, etc.). I guess the only thing it is really missing is integration with MS Word, but I could do without that. And it terms of connecting to library catalogs, you don't have to use those crummy Endnote connection files, that differ based on what system you are using (e.g., separate connection files for EndNote and CLIO, and every other database). Maybe we should just push Zotero as the citation management system....
