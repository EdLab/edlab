A Taste of SQL

  • 4 groups
  • At most 1 SQL maven per group
  • Pick one member from your group to write the queries in browser
  • SQL maven is not allowed to write queries, but can advise rest of group
  • 2 activities
  • Each group should navigate to its own linked SQL playground for the activity
  • Solve puzzles using SQL
  • Post solutions to blog before 11:00 am
  • Group with the most correct answers gets a special prize

Exercises

  1. Zena's favorite color?
    Solution:
    SELECT Name, Favorite_Color FROM Users WHERE Name = 'Zena';
    Solution is NOT "blue"
  2. Jamaal's birthdate?
  3. All users born after 2000?
  4. All users born in the 1990's?
