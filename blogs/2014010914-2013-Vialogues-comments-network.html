2013 Vialogues comments network

Sharon Hsiao

January 9, 2014

I looked at the user interaction network for all the comments in 2013. The first network visualization shows several clusters. We can easily see EdLabbers' cluster. And there are several other active user clusters last year too (It's a good sign that Vialogues are being really used by the community). Within our cluster, jagnitti and Katezvdp are the top 2 contributors (the darker the nodes indicating the more comments made). Other big contributors on Vialogues, made a lot comments, but did not seem to get a lot interactions back (norwoodfloydfan, malidogan etc). The second network visualization shows 2013 comments importance network. (the darker and bigger the nodes, the more important of the comments) We will see those big contributors were weighted down (if a user makes a lot contributions, but no interactions from the rest of the community, it will get weighed down).
