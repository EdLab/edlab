This Year’s Top Apps for the Arts

Renae Lesser

December 21, 2016


From theater to filmmaking, from music production to visual design, new digital tools are changing the way we create and learn about art. Here are a few of the most creative arts apps we reviewed this year.

1. The Shakespeare App
All the world’s a stage with this mobile e-resource, which includes the Bard’s complete works, study tools, and access to live performances.

2. Storybird
This storytelling platform for writers, readers, and artists of all ages makes it easy to incorporate visual elements into written work.

3. Flat
This sleek app allows users to write musical scores online and includes special tools for music educators.

4. Shadow Puppet Edu
This app makes it a breeze for students or novice filmmakers to create polished documentary films using only a smartphone.

5. How to Draw
Learn illustration skills with this mobile drawing app, which includes a database of easy-to-follow video tutorials.
