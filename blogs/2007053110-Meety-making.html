Meety-making

Anthony Cocciolo

May 31, 2007

A couple of folks have asked me how the Meety development is going and I thought I would share where I'm at with it. The major parts that are complete include:

  • Video and audio capture and storage
  • Transcription (which is pretty rough, but it does occasionally pick up some of the right words)
  • Front-end and back-end functionality to handle basic functions (logging, creating meetings, etc.) Here are some of the screens: Start Screen Login Screen Create/Load Meeting Main page Transcription dialog The parts that I will now be working on include:
    • Performing Semantic Analysis on transcript to create a set of statistically improbable phrases (will be used to query with)
    • Connection to Google Scholar
    • Front end tools for inter-person communication
    • Getting CIS to allow the Meety server to be a web server
