Data Vis Update: If you could see every Vialogue in one shot...

Janice Joo

March 21, 2013

Here's another update. I started with another idea (will post that process soon) but instead decided that the load-time was too slow... and came up with this one instead. This visualization gives you access to ANY vialogue ever created and still in existence (private or not). The brightness of the boxes represents how many comments are in each vialogue... Any vialogue with > 50 comments will be represented by the same level of brightness (50 comments = 100% blue)... In the future, it would be nice to give the user flexibility to - Select a range for how many comments there are... - Select a range for create-date (currently this visualization is organized in order of create-date - Filter by "Only Public" vs Private ... you get the point. This one makes more sense if you play with the live demo here. I implemented the same tooltip used in a previous experiment... But I have another front-end design idea in mind to implement... lots of creative potential here... Also, I need to clean up the JSON file a bit... so you might see reaaallly long video titles (those are errors).
