Name that app!

nolonger user

September 17, 2007

The alpha version of the new Survey Tool is nearing completion, but one key feature is still missing -- a witty name for the application! I've had some decent ideas so far, but it turns out a lot of them are already taken by random small companies / existing survey products. Here are some of the remaining ideas which don't conflict with existing sites:

  • Survey Sage
  • Survey Shark
  • EduSurvey
  • Survey Sidekick
  • Survey Widget
  • Survey Star
  • Survey Says

Please let me know if you like any of the above name(s). And if you can think of any more suggestions, I'd be extremely grateful! (as long as the name ideas aren't already in use -- a quick google search usually works) The name will be used to tie-in to the site's logo, branding, advertising, etc; so something that lends itself well to this (think: mascots?) would be great. The site also has a lot of educational content to teach survey-writers proper survey methodology, so names tying into the site's instructional nature are also a good bet. Thanks in advance!
