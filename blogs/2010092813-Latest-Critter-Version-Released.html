Latest Critter Version Released

fanwar

September 28, 2010

We've released a new version of Critter with many new improvements including: - New permissions model with an autocomplete UI that makes it easy to find and add people who you want to watch a video or discuss a video with. - Integration with Cloud storage to help us manage scalability of the tool in the future. - better help documentation to help you get up and running with using Critter. - Capture of more detailed user analytics. - Critical bug fixes from the previous version. Thanks to everyone who has been using and giving feedback on the tool so far. We are working to get another release out in the coming weeks. Many of the features we are prioritizing are the direct result of user feedback, including: - Survey/Poll driven comments. - Better UI feedback when you upload a video so you know what is happening. - Ability to give permissions to a discussion or video using a password/link that can be distributed via email or other channels by moderators. Look out for these and other improvements in the coming weeks. If you have a novel use for the tool that you have implemented or are thinking of implementing, please do share! Thanks to Zhou, Pranav, Joann, Megha and everyone else who has contributed!
