Latest Version of Critter released

pramnab1

October 22, 2010

We've released a new version of Critter with many new improvements including:

  • Improved User Experience:
    • Edit Video metadata i.e. Title, Description, Viewers.
    • Delete Video
    • Delete Discussion
  • Improved Privacy: Now Discussion moderators / Video owners can share their restricted videos or discussions via a link (like Google Docs) to other interested people.
  • Improved Learning Experience: Post survey comments (which could be single/multiple choice or free text).
Thanks to everyone who has been using and giving feedback on the tool so far. We are working to getting another release out in the coming weeks. Many of the features we have prioritizing are the direct result of user feedback, including what you all suggested:
  • Revamping the User Interface with better user look and feel
  • Improving performance for better user experience.
Look out for these and other improvements in the coming weeks. If you have a novel use for the tool that you have implemented or are thinking of implementing, please do share! Thanks to Zhou, Faisal, Joann, Megha, Pranay, Maitrey and everyone else who has contributed.
