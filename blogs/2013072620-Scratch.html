Scratch

Demetri Lales

July 26, 2013


Scratch teaches young aspiring coders the fundamentals of programming with fun abstractions like cartoons and blocks. To ease users into programming theory, lines of code are represented by blocks that can be connected to each other and stacked. Programmers can use these blocks to manipulate objects on a screen and create animations, videos, music, games, and interactive stories. Learning how these blocks of code work can teach students the logic behind computer programming and enable them to create their own media. The site allows users to view each otherâ€™s media creations and take a peek at the code used to create the media. The site also has some social functionalities such as commenting, liking, and a discussion forum.

Pros:

The platform makes it easy for users to test out and explore all of the script commands. Little to no writing is required to code with Scratch, which allows younger students to explore the platform and start learning about coding logic at a younger age. The transparency of the platform enables users to learn new coding techniques from one other. By looking at the different ways other programmers stack code, students can learn some new ways to arrange the blocks of code on their own creations. The content creation platform not only enables programmers to stack code but also add images, audio, and character figures.

Cons:

It would be beneficial for hopeful coders to be able to translate each block of code to an actual line in programming for some of the different languages. This feature might help users segue from novice programmer to real world coder.

Our Takeaway:

Scratch is an example of an online tool that takes the very complex practice of programming and simplifies it so young learners can practice and learn how to code. The platform has the potential to become a premier learning tool for future programmers. With mobile use and programming on the rise, the next clear step would be a mobile version of the platform. Mobile coding would give users an experience of coding that is directly applicable to their daily lives.

The Bottom Line:

Scratch gives budding programmers an environment to create anything their imagination desires.
