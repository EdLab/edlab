Link: Hero of the Code

Brian.Sweeting

April 10, 2014

*Private* Yesterday I was checking that NLT formatting and hyperlinks are consistent across different desktop and mobile browsers, and I thought I'd pass along two helpful Chrome extensions, if ever you should need them: 1) User-Agent Switcher for Chrome allows you to switch between different user agent strings and view a page in your browser as it would appear in: Chrome, Firefox, Opera, Safari, IE, iOS, Android, and Windows Phone. 2) Check My Links crawls your site for broken or dead links and reports what types of errors are occurring. If you ever notice an editing, formatting, or link error on NLT, please let me know and I'll either fix it or share it with the development team. Also, if you have any recommendations on extensions or apps for quality assurance, please feel free to pass along. Thank you for making NLT just a little more legendary.
