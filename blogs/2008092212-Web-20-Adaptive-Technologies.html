Web 2.0 Adaptive Technologies

fanwar

September 22, 2008

Lately, I've been scouring the web for different adaptive technologies that might be of use for some of the projects we are working on. I'm particularly interested in using these technologies for Survey Sidekick, which we hope will become a survey tool that intelligently helps its users create technically robust surveys. To this end, I came across one really interesting site called http://www.adaptiveblue.com. I was particularly impressed with their SmartLinks technology (http://www.adaptiveblue.com/smartlinks.html). The way this technology works is that you create a site with a bunch of links on it and AdaptiveBlue adds a small icon next to each link for the user. Then, the user can click that icon and get a popup showing all the relevant sites for the given link. So, rather than linking to just one specific site, you get links that guide people to a variety of different options they are interested in. What's especially cool about this technology is that it pops up suggestions based on the user's browsing history. So, if you shop from Amazon.com a lot and there is a link for the "Batman Begins" movie, then Adaptive Blue will suggest a link to buy a "Batman Begins" DVD from Amazon. It's not hard to imagine how a Smartlinks approach can be used to support adaptive online learning rather than e-commerce if that is the goal of a site like Survey Sidekick.
