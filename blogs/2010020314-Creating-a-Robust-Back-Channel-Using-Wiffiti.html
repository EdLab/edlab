Creating a Robust Back Channel Using Wiffiti

Adrienne Garber

February 3, 2010

In honor of Social Media Week in NYC, I want to share a meaningful social media application that encourages active participation in the back channel. Wiffiti 4 by LocaModa engages audiences by publishing real time messages to screens via mobile phones or the web, thereby creating cross-channel communication at said event. Event hosts create screens that subscribe to easily tagged content feeds. For example, a screen might display messages and pictures tagged "Adaptive Learning", "Teachers College", or "SECTtOR". The screenshot attached below was made using the "Make a Screen" tab on the Wiffiti website with the tag "EdLab" and a photo from Flickr (thanks, FGPferdt) for the background. There are many interesting uses of this tool to foster group engagement. Using the upcoming SECTtOR Design event as an example, a monitor display in the collaboration space at the EdLab could be embedded in other resources such as: the Gottesman Library homepage, the Teachers College homepage, on a devoted Facebook page, and on the EdLab's homepage, creating a communal audience experience fed by SECTtOR attendees as well as SECTtOR stakeholders following remotely on Twitter, Flickr, etc. SECTtOR attendees could be prompted to send a text message to the screen (e.g. @EVENT) in response to a speaker question, poll or other engaging activity. Click "more" below for additional features and a link to a whitepaper discussing this tool. Can you think of other uses within the EdLab for this type of interactive social media application? Please share! Other features of Wiffiti:

  • Does not require that the user is a member of any social network to participate.
  • Locations can also create unique mobile text backs (e.g. "Thank you for coming to our design workshop, please check our website for seminar snapshots.")
  • Wiffiti screens can be set to only display mobile messages.
  • There are controls for setting up G-Rated posts.
  • Mobile messages display a user's profile picture and custom name (if the user has registered with the system).
  • If the user is not registered with the system, the screen will display the user's randomly generated Wiffiti name.
    • For additional information, see LocaModa's whitepaper, "Building an Interactive Digital Out of Home Experience - Basic Considerations for Mobile & Social Interactivity on Digital Out of Home Networks". In it, LocaModa discusses the use of social interactive applications such as Wiffiti to create an enhanced user experience. LocaModa proffers that using interactive digital out of home ("DOOH") networks to stimulate user engagement results in a "pulse" for the event or location, which can then be used by the event planner to better target his or her message.
