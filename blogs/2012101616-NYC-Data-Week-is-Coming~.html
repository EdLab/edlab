NYC Data Week is Coming~

Sharon Hsiao

October 16, 2012

So glad to be in NYC :) NYC data week is coming. Oct 22-26. A lot of great events, FREE!!! Program details can be found HERE. Monday: mainly the developer talks. Tuesday: (I like the most) DataKind, Data Visualization. Data Gotham will present at night (I just registered, who's joining me?) Wednesday: a lot of demos, showcases. A lot meetups for dedicated groups after wednesday. Even a Hackathon on Friday.
