The Turing Tumble Teaches Kids How Computers Think

saraehardman

June 26, 2017


A new toy gives kids insight into the inner workings of computers by having them build mechanical computers powered by marbles. The Turing Tumble, named after computer scientist Alan Turing, was created by Paul Boswell. While teaching at the University of Minnesota, Boswell noted how important it was for all students to learn how to code and have a solid understanding of computers.

Although there are many coding and programming toys that are available now, many donâ€™t dive into the actual mechanics of computing. "They all treat computers like abstract boxes," says Boswell. "They teach how computers behave, but not how they work." The Turing Tumble is a mechanical computer that kids build themselves; by playing with it, they learn the secrets stored in computer chips. Using the six different pieces of the toy, kids can solve logic puzzles, compute simple equations, and create patterns. Boswell included a set of 51 puzzles that kids can work through to become master logicians. The Turing Tumble is currently raising financing on Kickstarter and has already far exceeded its fundraising goal.
