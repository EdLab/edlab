Portable Applications in the Library

gnatriello

July 13, 2010

This recent column on portable applications carries some interesting implications for our efforts to help library patrons use library computers in more flexible ways. Because we have to maintain hundreds of computers for public use, we are often forced to standardize and simplify the resources we make available on library machines. Patrons, however, bring the same expectations they have developed in the course of using personal machines to their work in the library. It occurs to me that portable applications might allow patrons to bring their individual tools on a small usb drive for use with library computers. If further investigations confirms this assumption, then we might consider offering a library workshop to teach patrons about portable applications and their use in the library and elsewhere
