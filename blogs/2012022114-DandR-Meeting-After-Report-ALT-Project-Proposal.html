D&R Meeting After Report: ALT Project Proposal

Demetri Lales

February 21, 2012

PRIVATE PLEASE First off I'd like to thank all of your for hearing out my proposal and providing some great feedback. There was a lot of great ideas for features on the site that would house the ALT database. One great idea was the inclusion of information about the companies that produced the ALT product. Features would include a company profile, and contact information. I wasn't sure if the database would be entire useful for the entrepreneur community but now it seems that they, as well as developers, can be a potential audience for this product, in addition to educators, students, and researchers. Developers can inform us on any updates on their ALT products so that it's up-to date. A great reference site that was suggested was Cnet.com, a site that give detailed reviews for a lot of the technology we use everyday. They have a section dedicated to user reviews, which is an idea that would be great for ALT. This can give the site a social component that it seemed to be lacking before. There were many more suggestions that were great for the ALT site, check out the pictures below and see what each group suggested. I'll continue to work on this project, mostly by building up our ALT database. If any of my fellow Edlabbers want to help me fill up the database, with any applications that you believe is an ALT, then contact me! I'll share the Google Doc with you and then you can add any application that you think deserves to be in the Adaptive Learning database.
