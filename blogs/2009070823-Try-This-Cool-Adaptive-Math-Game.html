Try This Cool Adaptive Math Game

Ting Yuan

July 8, 2009

Check out this COOL arithmetic game named Active Quiz developed by a game designer in California. It showed me a record graph when I completed playing the game. My score was rated as top 1% among players within the same age group! What about you? Have a try! There are some "secrets" hidden in the equations during the game playing. Let me know what you may have found! Its adaptive feature lies in that the speed and result of your calculation would continuously influence the next group of equations shown up. Would this game give us some enlightenment for EdLab's future game development? How would we want to present and organize players' assessment records upon the completion of their game playing? Considering the iPhone math game currently under prototyping, do we need to have a database set up for the adaptive part? Or as Pranav suggested that all the adaptive functions might be realized through coding in OPEN GL?
