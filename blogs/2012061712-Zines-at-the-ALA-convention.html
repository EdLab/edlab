Zines at the ALA convention

Anne Hays

June 17, 2012

As the New York Times noted last year, zine-making is on the rise, as evidenced not only by the number of new zines in circulation, but by the number of academic libraries starting zine collections. Zine libraries in the New York City area include:

  • Barnard
  • Brooklyn College, CUNY
  • NYU's riot grrrl collection
  • and the independent non-profit ABC No Rio zine library
Therefore, it is only fitting that this year's American Library Association (ALA) convention is hosting a number of zine-related events, including a raffle drawing, a panel on starting a library collection, a panel on how to make a zine, and one on how to use zines in the class room (see my own earlier post on that topic).
