Coding Toy Robots A Boon for STEM

DebraLee

November 6, 2017


For some, learning to write code can be downright boring, but this robot kit teaches kids to code as they create. Algobrix encourages children ages 5-13 to imagine their own robots or build one from a design template (AlgoMonster, AlgoBuddy, AlgoElevator) provided in the kit.

Activity cards suggest different animated sequences for the robotsâ€™ movement and coding blocks replicate aspects of computer coding, including parameters, loops, and functions. Actual learning occurs when the child arranges the Lego-compatible coding blocks in a particular sequence and then observes the robot performing functions like moving, lighting up, and making noises associated with the sequence.

The game includes step-by-step coding missions and blocks labeled with symbols. Algobrix hopes the kit will teach children basic science, technology, engineering, and math skills. How do you think games like Algobrix encourage STEM learning? Join the discussion on Vialogues.
