Matchbox and the Digitization of the College Admissions Process

Demetri Lales

May 21, 2012

Matchbox is iPad application that takes the college application process and makes it paperless and stream-lined. Matchbox is designed to cut the time it takes individuals in charge of college recruitment to fully review an applicant. What is interesting about this application is that it fully digitizes a paper-filled process, and makes the process more efficient. As its website says Matchbox allow their users to “makes better decisions faster” Features:

  • Scoring- This allows users to score the applicant based on a number of criteria, and attach evidence to these scores. This feature allow users to post questions that interviewers would use for when the applicant is interviewed.
  • Annotation- Texts from essays, recommendations, and resumes can be highlight, selected and connected to a score, color-coded, etc.
  • Roles- Matchbox allows users to assign and customize roles. This is to match how the application process is like in real life, diverse and with multiple individuals involved. Some examples include: screener, reader, interviewer, career interviewer, academic reviewer, director, etc.
  • Assignments- This feature allows the user to streamline the task of assigning application to reviewers.
    • Our Takeaway: Judging by the collaborative features it seems that if a college wants to use the Matchbox application it would need to have multiple iPads. While the product can be free there are some priced options that includes more features, quick support and implementation help when colleges first purchase the product. Once colleges get past these hurdles they can start streamlining the college application process, but one has to wonder if this will save colleges money. One can argue that since Matchbox saves time it saves money as well. This is a good example of creating a product to solve a very specific problem, and fully digitize a paper-filled process. Will there be more products on the horizon that will the same for other paper-filled tasks?
