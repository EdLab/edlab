Docdel system and first few weeks at EdLab

Xiaobin Zheng

March 5, 2010

Hello everyone, I'm new guy here at the EdLab and I'm going to tell you about what I have been working on so far here at the lab. During the past few days, Ankit and I have been doing some work on the Docdel system, which provides students and teachers with a new way of requesting and posting study materials. Combined with the ClassWeb system, it is a good place for students and teachers to share materials. However, Docdel relies on some data in the databases of ClassWeb, which is going to be shut down in the near future. Our goal now is to build back-end databases for Docdel to let it keep working after ClassWeb is shut down. In addition, we also want to add a function for course session management. This will perform as a similar function to the one currently being used in ClassWeb. With this two changes, the Docdel system would be an independent system that does not need to use resources from other places, namely ClassWeb, and would be much more convenient for teachers to manage course information. Here is how the current system works and what session management is like: -When someone creates or deletes sessions on ClassWeb, session appears/disappears from the Docdel automatically; -Either a teacher or a student can make a request for readings from Docdel; -One can assign readings to a certain session on Docdel and the readings will appear automatically on a ClassWeb session. Readings uploaded directly on ClassWeb won't appear on the Docdel system. The new system, as described above, needs to allow a teacher creating or deleting sessions to do so directly on Docdel instead of logging into ClassWeb. What we have done by now are as follows: -Finding out the specific data used by Docdel from ClassWeb; -Designing new tables in the databases of Docdel and importing all needed data from ClassWeb; -Designing the prototype of new UI for session management; With Ankit's help, I'm currently working on the new UI and the modification of the system files and I hope that in the next few weeks, we would be able to finish this project. Hope everything goes well with everyone and it's really great to be a member of the EdLab!
