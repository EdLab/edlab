Next Version of Vialogues out!!

pramnab1

July 11, 2011

The Vialogues team is proud to announce the release of an updated Vialogues with new features and enhancements. Some of the new features included in this release are: - New User Interface. - Notification. - New look for embedded vialogues. Enhancements: - Moderator comments are identified. - Editing of comments are allowed - Emails send when users comments. We're looking forward to people using these and other features in a variety of educational/commerical settings. Thanks to the entire Vialogues team in getting this release out: Gonzalo, Amelia, Josh, Alex, Wei, Zhou, Yudan, Joann, Kate, Joyce, Hui Soo and others. Some other awaited features which are in the pipeline for the upcoming release are:- 1) Full Text Search with Tag Categorization 2) Performance - User interface and backend. 3) Make Vialogue Interfaces more consistent. Enjoy and keep the feedback coming!
