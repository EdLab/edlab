Trends in Ed: Moneyball Approach to College

Demetri Lales

December 22, 2011

According to an article in the Chronicle colleges are starting to depend on data to help make decisions like who students will pair up with for group discussions. One web application used to make these decisions based on the data inputted is Learning Catalytics. In Austin Peay State University, they are developing a software that will help students pick a major using data like a student's grades. Yet another program is becoming popular that uses data to help students make important academic decisions. ConnectEDU is a new site that helps students decide what college best fits them, this can lead to student's not needing to go through the process. There seems to be a growing trend in using student data to help students make academic decisions, and help professors with classroom activities Classroom data mining is becoming more popular, in a community college in Arizona it is being used to predict with a 70% accuracy whether a student will score a C or higher in a course. This is made possible by putting cameras over every desk and observing which sites a student views, thus observing their studying habits. This information can help teachers decide which students need extra help. Using this type of data analysis can help improve a student's educational career, but there is the concern of whether or not all this data mining is going to invade a student's privacy.
