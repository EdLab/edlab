Light Table - Another Way to Learn How to Program

Sharon Hsiao

August 28, 2012

Light Table is a recent Kickstarter funded project, proposing a new kind of Integrated Development Environment (IDE) - for the creation and exploration of programs. A few guiding principles of Light Table:

  • Never have to look for documentation
  • Files are not the best representation of code, just a convenient serialization.
  • Editors can be anywhere and show you anything - not just text.
  • Trying is encouraged - changes produce instantaneous results
    • We can shine some light on related bits of code
    Over the interview, the cofounder mentioned a lot of organizations said it would be the most amazing teaching tool for programming that's been around. Full story here. I actually find it indeed interesting. For programming novices, it's already super overwhelming to learn a complete new language, the syntax, the programics and the semantics. Not to mention all the softwares, applications etc. A simpler, yet, an intelligent and supportive tool is definitely demanded. Even for advanced programmers, they will search for documentations, or examples from time to time. I did a small project before, it's a web IDE by proving programming examples from several sources (wikibooks, youtube etc) based on your current codes. Here's the paper Educational Social Linking in Example Authoring. I think the idea is simple, similar to Light Table, but more “social” (cheap, less AI approach)-oriented. I'd really like to see how Light Table is being used in programming language learning.
