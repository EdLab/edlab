Learnstream

Jenny Shen

November 11, 2015


With the wealth of online resources available for learning a new skill, concept, or language, it can be overwhelming to keep track of all the relevant websites available. Learnstream was created to keep all these resources organized—it's an open source tool for users to sort their sites and serves as a network of guides made up of earmarked web pages.

Pros:

One of Learnstream’s features that makes it conducive to learning is its open source model, which allows anyone to put together a guide and share it on the Learnstream site. This makes learning resource curation a collaborative process. It also eliminates the redundancy found on sites like reddit, where several threads list many of the same websites. And a tagging system keeps all of the site’s guides organized for easy perusal.

Cons:

As Learnstream is relatively new and hasn't seen much traffic, there aren't many established guides on the network (only 66), with most of them on the topic of physics. This is a natural problem with open source tools, which is usually overcome when the site hits a tipping point of virality—the more people who use the site, the more likely it is to attract new users.

Some people might also prefer to use their browser's bookmarking feature instead of a third party tool to keep track of their learning resources. This could prevent Learnstream’s from becoming widely used. And it doesn't help that users cannot save resources that aren't in a web page format (no documents, videos, or audio files).

Our Takeaway:

Overall, this site tackles the problem of information overload in online learning. Making online resource curation a collaborative process is helpful, but Learnstream may need to offer additional features to make it valuable enough to attract new users, which is important for making the open source model itself sustainable.
