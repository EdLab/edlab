Lets go Jogging ... on the Web

Stephen Asunka

August 27, 2008

"Jog the Web" is a web-based tool that allows anyone to create a sequential guide to a series of websites. It is a free resource and I can see it has a great potential in education as it allows the organization of information in a meaningful way, thus ensuring purposeful access of web resources. There is also an opportunity for users to add comments etc. I can envisage teachers using this tool to create a sequence of websites that guide students progressively through some learning or information seeking processes. I am in the process of creating an "Adaptive Learning Resources" Jog (or track). But see what I have created for a trial. Simply click on the animated blue arrow to jog through the four most fantastic (!) websites I discovered on the web.
