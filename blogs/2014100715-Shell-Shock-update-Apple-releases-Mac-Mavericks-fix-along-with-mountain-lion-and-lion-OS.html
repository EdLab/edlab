Shell Shock update: Apple releases Mac Mavericks fix along with mountain lion and lion OS

pramnab1

October 7, 2014

Please download the fix accordingly for your Mac OS X: If you want to find out which Mac OS X you have, click on the apple icon on the top left corner of the screen and go to "About this mac". You will see a number like 10.x.x based on the first x number you can download the below package for addressing the issue: 1) Maverick: 10.9 2) Mountain lion : 10.8 3) Lion: 10.7
