EdlabCAS and it's beta launch with Critter

pramnab1

April 20, 2010

For the past couple of months, EdLab's Development & Research team has been working on setting up a Central Authentication Single Sign On solution for EdLab Applications to improve user experience and enable data sharing between applications for more personalized recommendation. Team: Jo Agnitti, Hui Soo, Pranav and Faisal. It gives us great pleasure to announce that we are launching our initial release with Critter, which would enable users to get authenticated through CAS. It's still in a early stage and we would be building more features to it soon. We would appreciate it if you can point to any issues you face while using EdLabCAS and Critter to us so that we can fix up those issues. For those of you who would like to create a EdLab CAS account to access Critter and other new services from EdLab in the future, please follow the instructions below. To get the EdLab CAS account for using the Critter Application, we need you to send us the following information in an email to Pranav (pranav.garg007@gmail.com) or Faisal (fanwar@gmail.com). 1) Email Address - would be your username. 2) FirstName: 3) LastName: 4) Backup Email address is optional (It would be later used for setting up the forgot password utility) 5) Date of Birth: Please give us a day to get back to you with a confirmation email of setting up the account correctly and a random password which you can use for logging in. Eventually there will be an automatic user account creation page. However, until we get that setup please send an email with the requested details. We would be looking forward to hear your suggestions/feedback and any issues you encounter.
