CSS 3 is so cool!

Ellen Qian

April 9, 2010

I am working on TLN UI design right now. When I first came up with jQuery, which had a perfect MIT license, I found that its UI plugins had a strong and amazing vision effect. However, I got a brief view about CSS 3 yesterday and I was surprised to see that CSS could do great great work. Although CSS 3's vision effect must be supported by your browser and W3C does not make all of the tags defined in CSS 3 as a standard. It is still an excellent UI design tool as it combines javascript and div together into one tag. Here is the demo. P.S. I love that falling leaves and coke advertisement. They are made of PURE CSS!!
