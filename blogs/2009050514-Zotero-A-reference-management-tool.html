Zotero: A reference management tool

fanwar

May 5, 2009

Recently, I had to shift away from Endnote and was searching for a good alternative to use for managing citations/references for research. First, I found this wikipedia page that is really useful for comparing everything that is out there. I decided to use a Firefox plugin called Zotero to do my reference management and have been pretty happy with it. Zotero is a product of work done at George Mason University to support library service tools. It integrates with your web browsing so you can save snapshots of web pages that you visit in addition to creating citations of them. It also lets you download citations from many cites and does this automatically for some (like the NYTimes). It even lets you create a timeline of the references you created so you can visually see how scholarship has progressed in a particular set of citations. Finally, you can sync it with an online repository, so you can access your references regardless of what computer you happen to be using. Best of all ... IT'S FREE!!! You can check it out at: http://www.zotero.org/
