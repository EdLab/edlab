Microsoft Unveils 'Bing' Search Engine

Stephen Asunka

June 4, 2009

Microsoft has released Bing - a successor to its Live Search Service. The company is calling Bing a decision engine rather than a search engine, as it includes query intent recognition and document summarization technologies, as well as a new user experience model that "dynamically adapts to the type of query to provide relevant and intuitive decision-making tools".

Bing also includes various pre-configured search categories such as shopping, travel, healthcare, live flight data, a five-day weather forecast etc. Why the name Bing? I suspect Microsoft is hoping that the name will catch on and ultimately become an internet buzzword. So from now on, don't just google, also bing as well. The Search Engine War is clearly on.
