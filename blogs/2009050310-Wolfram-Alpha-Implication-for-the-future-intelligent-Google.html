Wolfram Alpha -- Implication for the future intelligent "Google"?

Ting Yuan

May 3, 2009

Wolfram is the famous creator of Mathematica, a powerful software which can help to solve math problems in Greek characters (just like what a mathematician will do for you). The software is playing a leading role in symbolic calculation, which have been used in multiple popular math softwares (e.g., Matlab) today. Wolfram Alpha is a computational knowledge search engine under development by Wolfram's group, which is anticipated to "answer" search questions instead of giving pages of links like what the main search engines do today. For example, you can search "what's the french population during 1940-2008". It will show you the curve of the French population in those years. Then if you further ask "How many babies born in France every year?", and if it can't find the answer to this question directly, it will try to search the result of "french birth rate", then use the population and birth rate to generate the result for you through calculation. Intelligent machine learning will bring major changes to various fields related to human beings. So I guess we can imagine what web 3.0 will look like after web 2.0 featuring in interaction among users. Can web 3.0 become our individual personal web robot which can not only help us to search the results we want but also generate results or even give "advices" if the results are not directly found? I look forward to the release of Wolfram Alpha later this month.
