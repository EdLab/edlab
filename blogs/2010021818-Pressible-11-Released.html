Pressible 1.1 Released

Patrick Carey

February 18, 2010

This afternoon the Pressible team released Pressible 1.1. If you are using Pressible you should see the changes right now. The majority of the changes that were made were design-based or affect the code behind the scenes. We think this release makes the Pressible Theme look and function much more consistently. Some of the key changes made in this release:

  • The addition of a Search function.
  • Brightcove embed support for After Ed.
  • The header now includes 5 image boxes, each box will display a cropped piece of an image that is uploaded to the site (and not attached to a post -- well, at least that's how it turned out to work), this is another foray into extending the publishing philosophy of Pressible to the actual design of the site.
  • Timestamps on posts and comments.
  • Restyled header, page links, background, and font styles.
  • Separate drop-down menus for Recent Topic subheader.
  • "Chat bubble" styling for commenting (style still evolving).
  • The theme is styled differently for a "single-user" Pressible site.
  • Pressible now allows attaching these file types: jpg, jpeg, png, gif, mp3, m4a, pdf, doc, docx, txt, xls, xlsx, ppt, pptx.
We think this release strengthens the initial Pressible theme and builds a stronger foundation for the future. We are excited that we were able to make so much progress in a relatively short amount of time. If you have not signed up for a Pressible site I would encourage you to do so.
