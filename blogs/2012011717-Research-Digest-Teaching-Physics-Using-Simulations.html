Research Digest: Teaching Physics Using Simulations

Demetri Lales

January 17, 2012

Wieman, C.E., Adams, W.K., Loeblein, P., & Parkins, K.K., (2010). Teaching Physics Using PhET Simulations. The Physics Teacher, 48(4), 225-227 Description: This paper describes how a physics interactive simulation application called PhET can be integrated in the classroom. The authors also discuss how simulations in general can be woven into the classroom curriculum and generate effective teaching. Some of the major advantages, according to the authors, to using this type of simulation application in the physics classroom include: simulations can be used in classrooms where the real equipment is either not available or impractical to set up, they can be used to do “experiments” that are impossible to do otherwise, and that it is easy to change variables in response to student questions that would be difficult or impossible to change with real apparatus. It is stated that teacher guidance is very necessary when learning with simulation as teachers can pose a scenario, ask for students' prediction, then run the simulation and discuss what happened in terms of physics. There are brief descriptions on how to use the simulations for homework and lab activities, as well. EdLab Relevance: After looking into one of the PhET, namely the Capacitor Lab, one can see that the simulation is very interactive with many options and can visually represent how a capacitor works pretty well. As the paper says, even with the simulation, students would still need a teacher's guidance to explain what exactly is happening when students play around with the different options (increasing decreasing voltage, plate area, and plate separation). So the simulation is a great tool to use in the classroom, when the actual equipment is not available, but it will not teach students about the principles of physics on it's own. The authors believe that using a simulation in the classroom will help address common misconceptions about some physics concepts. This is an important part of teaching any STEM subject, as many students that form their own conclusions about some scientific principles, are usually incorrect.
