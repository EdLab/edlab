Adaptive Learning Slideshow!

Demetri Lales

February 29, 2012

While doing some market research and building up the Adaptive Learning Technology (ALT) Database, I bumped into this very informative slideshow from Cogbooks that can better explain what adaptive learning is. Check it out here! CogBooks is definitely going to be added to the database since it's a cloud based adaptive learning application for learning more about business skills. We are finding more ALT apps for higher education learning which is making things pretty interesting. I hope the slideshow better explains adaptive learning, I'll be to post more interesting findings as this ALT database grows!
