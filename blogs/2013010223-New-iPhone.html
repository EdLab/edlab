New iPhone?

Francisco.Mendoza

January 2, 2013

It is being rumored that Apple is testing their new mobile device, the " iPhone 6.1 ". It is suppose to run on a new mobile operating system. All of this is just a rumor but there could be some truth to it due to the activities noted in logs from Apple's headquarters. This would be an ideal start to the year for Apple, as they finished 2012 in great position along with Android by having 17.4 million new activations in a single day. There have already been rumors about Apple making an iPhone 5s which will apparently have minor changes to the latest iPhone. This seems to be the normal marketing plan for Apple as it usually makes small changes to their mobile devices and put it out for retail instead of simply upgrading and changing any bugs or adding new features. Will people become tired of small changes or continue waiting hours in line for Apple's latest phone? Should Apple stop releasing phones frequently and use that time to create an ultimate phone that would be worth the upgrade?
