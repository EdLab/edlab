EdLab Seminar (STAFF ONLY) - Tool Design

Mar 8, 2018 12:00 PM - Mar 8, 2018 1:30 PM

At this seminar we will develop designs for a Learning Theater column tool set.

Topic: Tool Design

Description: We will work collaboratively to develop a set of tools that might be affixed to the columns in the Learning Theater.
