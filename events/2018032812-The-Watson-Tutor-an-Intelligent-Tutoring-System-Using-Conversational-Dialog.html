The Watson Tutor: an Intelligent Tutoring System Using Conversational Dialog

Mar 28, 2018 12:00 PM - Mar 28, 2018 1:30 PM

We describe the Watson Tutor, a personalized, conversational system that is designed to improve a student's engagement with and mastery of a set of learning content using a Socratic dialog technique. The Tutor uses natural language to communicate with the student, attempting to mimic the tone, feedback and responses of a good human tutor. The talk will cover the design of the Tutor's dialog system and describe how some of the learning activities supported by the Tutor work behind the scenes. The Watson Tutor is today presented as a chatbot embedded in digital courseware sold by Pearson Education. It is currently being piloted by 100s of students nationwide, and is planned to be sold as a commercial offering in the near future.
