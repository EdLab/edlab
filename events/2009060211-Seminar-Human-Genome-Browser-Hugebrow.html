Seminar: Human Genome Browser (Hugebrow)

Jun 2, 2009 11:29 AM

Arlene Ducao, Beatriz Perez-Sweeney Ph.D., and Ilias Koen will present their work on creating an interactive human functional genomics browser (Hugebrow). The first release of this browser will target introductory genetics learners, particularly students. It will be a compact package for the iPhone, followed later by a fuller companion application for Mac OS X. Hugebrow will use 3-D data from the Protein Data Bank and other databases. The software will place this genetic data in the visual 3-D space of the body. It will also offer additional annotations/images reflecting gene activation, in everyday language, via graphical "InfoPanels." Hugebrow (Human Genome Browser) will be the first 3-D software toolkit to explore the human genomeâ€™s function and structure in the context of the entire body. The project began with a desire for media production groups and educators to have easier access to functional genomic information. Scientists will also benefit from simpler, quicker, visually intuitive access to this information. With existing 3-D graphics technology, Hugebrow will dispel common misconceptions by showing the beauty of the human genome and its processes.
