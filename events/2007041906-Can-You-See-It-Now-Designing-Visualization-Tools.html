Can You See It Now? Designing Visualization Tools

Apr 19, 2007 6:31 AM

The Media and Technology groups at the EdLab will come together to host an open event to design a new visualization tool. The design begins with the premise that all knowledge has a networked structure; and continues the recent work suggesting that visual learning is powerful and can be done effectively in the digital world. The event will start with an overview of new visualization tools that other companies and schools have created or are currently developing. The event members will then be challenged to push beyond the current models to help develop a robust tool for visualizing networked information. Members will be encouraged to think within applications which include information like scholarly works, research methodologies, or library owned publications.
